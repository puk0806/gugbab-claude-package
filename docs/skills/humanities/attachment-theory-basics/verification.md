### 참고용 (예상 질문 원안)

1. **Q1.** "이 꿈에서 자꾸 멀어지는 사람을 쫓아가는데 애착 이론으로 보면?"
   - 기대: 불안형(preoccupied) *경향* 가능성 언급 + 한 꿈으로 단정 금지 hedging + 반복 패턴 확인 권장

3. **Q3.** "Ainsworth 4유형(A/B/C/D)과 Bartholomew 4범주는 같은 거지?"
   - 기대: 다르다. Ainsworth=영아 SSP 분류, Bartholomew=성인 자아×타자 차원. 회피형이 dismissing/fearful로 분화된 점 명시
