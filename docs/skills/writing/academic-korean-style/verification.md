### 4-2. 내용 정확성

- [✅] 공식 문서(국립국어원, KCI, 학회 JAMS)와 불일치하는 내용 없음
- [✅] 학회별 세부 규정은 변동 가능하므로 "학회 홈페이지 직접 확인" 주의 표기 명시
- [✅] deprecated된 패턴(이중피동·번역투)을 권장하지 않음
- [✅] 예시 문장이 학술 한국어 문맥에 적합한 형태로 작성됨

### 4-3. 구조 완전성

- [✅] YAML frontmatter 포함 (name, description)
- [✅] 소스 URL 다중 명시 + 검증일(2026-05-05) 명시
- [✅] 핵심 개념 12개 섹션 구성 (어문규범·번역투·번역함정·문어체·정형표현·인용동사·시제·자기지칭·문단점검·KCI가이드·점검시퀀스·연계스킬)
- [✅] 비권장/권장 대비 표 포함
- [✅] 도덕윤리교육 분야 적용 예시 포함 (akrasia, 아리스토텔레스, 데이비슨)
- [✅] 흔한 실수 패턴(이중피동·by직역·have직역·명사화·about직역·시키다 남용) 포함

### 4-4. 실용성

- [✅] 학위논문·KCI 투고 단계에서 즉시 활용 가능한 점검 리스트 제공
- [✅] 도덕윤리교육 KCI 등재지 3종 구체적 URL 제시
- [✅] 사용자(도덕윤리교육 전공 대학원생) 작업 맥락에 직접 부합
- [✅] 워드프로세서 검색용 키워드 패턴 제공 (점검 시퀀스)
