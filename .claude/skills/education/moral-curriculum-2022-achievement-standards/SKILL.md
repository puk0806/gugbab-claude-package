| 2026년 3월 1일 | 초등학교 5~6학년, 중학교 2학년, 고등학교 2학년 |

> 출처: 교육부 고시 제2022-33호 부칙(시행 일자) 및 교육부 안내 자료. 도덕과는 초등 3학년부터 정규 과목으로 편성되므로 실제 도덕 교과 적용 시점은 **초등 3·4학년 = 2025년 3월부터, 5·6학년 = 2026년 3월부터, 중학교 = 2025년부터 단계, 고등학교 = 2025년부터 단계**.

### 1-2. 2024년 부분 개정 사항

> 주의(미확인): 2024년 도덕과 부분 개정 고시 여부와 구체적 변경 내용은 본 검증 작업에서 NCIC 원문으로 확정하지 못함. 인용 전 NCIC(`ncic.re.kr`) 또는 교육부 고시 게시판에서 도덕과 관련 후속 고시(예: 제2024-XX호)가 추가되었는지 직접 확인 필요. 부분 개정이 있으면 별책 6의 일부 항목이 갱신되었을 수 있음.

## 2. 교육과정 구조 (영역·학년군 매트릭스)

2022 개정 도덕과는 학교급 전체에 걸쳐 **4개 영역**으로 구성된다.

| 영역 코드 | 영역명 |
|-----------|--------|
| **01** | 자신과의 관계 |
| **02** | 타인과의 관계 |
| **03** | 사회·공동체와의 관계 |
| **04** | 자연과의 관계 |

> 주의: 영역명의 정확한 공식 표기(예: "사회·공동체와의 관계" vs "사회공동체와의 관계", "자연과의 관계" vs "자연·초월과의 관계")는 학교급별로 미세하게 다를 수 있음. 본 스킬은 검색된 다수 소스에서 통용되는 표기를 사용했으나, **학위논문에 영역명 자체를 인용할 때는 NCIC 원문에서 정확한 표기를 다시 확인**할 것.

### 학년군별 코드 체계

| 학년군 | 코드 접두사 | 비고 |
|--------|-------------|------|
| 초등 3~4학년군 | `4도` | 예: `[4도01-01]` |
| 초등 5~6학년군 | `6도` | 예: `[6도01-01]` |
| 중학교 1~3학년 | `9도` | 예: `[9도01-01]` |
| 고등학교 일반선택 「현대사회와 윤리」 | `12현윤` | 일반선택 과목 |
| 고등학교 진로선택 「윤리와 사상」 | `12윤사` | 진로선택 과목 |
| 고등학교 진로선택 「인문학과 윤리」 | `12인윤` | 진로선택 과목 |

> 주의: 고등학교 과목명·접두사는 다수 소스(나무위키, 교육부 안내문, KCI 논문)에서 동일하게 확인되었으나, **세부 성취기준 코드 번호 체계와 영역 구분은 NCIC 원문에서 직접 대조 권장**.

## 3. 성취기준 전문(全文) — 학년군별

> 표기 규칙
> - **`✓ 검증`**: 2개 이상 독립 소스에서 동일 문장이 확인된 항목 (학위논문 인용 가능)
> - **`[NCIC 원문 직접 확인 필요]`**: 본 검증 작업에서 전문 텍스트를 PDF 원본으로 확정하지 못한 항목. **추측 금지** — 인용 전 NCIC 원문 직접 대조 필수.

| `[4도01-02]` | 정직의 의미를 알고 모범적인 사례를 탐색하여 바르게 행동하려는 태도를 기른다. | ✓ 검증 |
| `[4도01-03]` | 성실한 생활의 모범 사례를 탐색하고 시간 관리를 위한 생활을 계획하여 지속적인 자기 성장을 모색한다. | ✓ 검증 |
| `[4도01-04]` | 다른 사람의 관점을 수용할 수 있는지를 도덕적으로 검토하고 도덕규범을 내면화하여 도덕적으로 행동할 수 있는 자세를 기른다. | ✓ 검증 |

#### 영역 02 — 타인과의 관계

## 4. akrasia(의지의 약함) 논문에서 활용도 높은 성취기준

> akrasia 논문은 인지(앎)–정의(욕구·감정)–행동(실천) 사이의 괴리, 도덕적 의지·자기 조절 약화를 다루므로 **「자신과의 관계」 영역의 도덕적 의지·자기 조절·자기 성찰** 관련 성취기준이 핵심 인용 대상이 됨.

### 4-1. 인지–정의–행동 통합과 직접 연결되는 성취기준

| 코드 | 핵심 키워드 | 활용 포인트 |
|------|-------------|-------------|
| `[4도01-04]` ✓ | "도덕규범을 내면화하여 도덕적으로 **행동할 수 있는 자세**" | 앎과 행동의 연결, 내면화 단계 |
| `[6도01-01]` ✓ | "자신의 생활계획을 세우고 **실천**하여 **주체적인 삶의 태도**" | 의지 형성과 자기 주도성 |
| `[6도01-02]` ✓ | "생활 습관에 대한 **성찰**, 자기 생활을 점검" | 자기 조절·성찰 |
| `[9도01-01]` ✓ | "자신의 삶과 가치관에 대한 **성찰**, 도덕이 필요한 이유에 근거하여 **도덕적인 삶에 대한 의지**" | akrasia 핵심 — 도덕적 의지 형성 |
| `[9도01-05]` (미확정) | "내면 성찰, 심리적 고통·불안·우울 원인" | 정의(情意)적 차원의 자기 조절 |

### 4-2. 초등 4학년군의 도덕적 의지·자기 성장

`[4도01-03]` ✓ — "성실한 생활의 모범 사례 탐색, **시간 관리**를 위한 생활 계획, **지속적인 자기 성장**" → akrasia 극복의 초등 단계 표현으로 인용 가능.

## 6. 2015 개정 vs 2022 개정 비교 (혼동 방지)

### 6-1. 동일 코드 ≠ 동일 전문

**중요:** 2015 개정 도덕과 교육과정도 [별책 6]이며 동일한 코드 체계(`4도01-01` 등)를 사용한다. 그러나 **전문(全文)은 다를 수 있으므로** 인용 시 반드시 어느 개정인지 명시해야 함.

| 항목 | 2015 개정 | 2022 개정 |
|------|-----------|-----------|
| 고시 번호 | 교육부 고시 **제2015-74호** [별책 6] | 교육부 고시 **제2022-33호** [별책 6] |
| 고시일 | 2015.09.23. | 2022.12.22. |
| 영역 구조 | 4영역 (자신/타인/사회·공동체/자연·초월) | 4영역 (자신/타인/사회·공동체/자연) — 표기 미세 차이 가능 |
| 성취기준 작성 틀 | 지식·이해 중심 | **지식·이해 + 과정·기능 + 가치·태도** 3차원 |
| 핵심 아이디어 | 미도입 | **핵심 아이디어** 도입 |

### 6-2. 비교 예시 (동일 코드 다른 전문 위험)

`[4도02-01]`의 경우:
- **2015 개정**: "가족을 사랑하고 감사해야 하는 이유를 찾아보고, 가족 간에 지켜야 할 도리와 해야 할 일을 약속으로 정해 실천한다." (검색 다수 일치)
- **2022 개정**: 유사하나 후반부가 다를 가능성 있음 → **NCIC 2022 원문 직접 확인 필수**

> 학위논문에서 "가족" 관련 [4도02-01]을 인용할 때 2015 전문을 2022로 잘못 표기하면 결격 사유가 될 수 있음.

### 6-3. 핵심역량 변경

> 주의(미확인): 2022 개정에서 도덕과 교과 핵심역량(자기 존중 및 관리 능력, 도덕적 사고 능력, 도덕적 대인관계 능력 등)의 명칭·개수가 2015 대비 어떻게 변경되었는지는 KCI 논문(예: "2022 개정 도덕과 교육과정의 교과 역량에 대한 제고")에서 다루어지나, 본 검증 작업에서 정확한 변경 사항은 확정하지 못함. 인용 전 NCIC 원문 "2. 교육과정 설계의 개요" 또는 "교과 역량" 절 직접 확인 필수.
