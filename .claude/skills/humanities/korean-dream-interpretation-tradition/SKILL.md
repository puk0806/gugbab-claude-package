## 1. 한국 해몽 문헌의 역사

### 1.1 정의 (한국민족문화대백과사전, 항목 E0011278)

| 용어 | 정의 |
|------|------|
| 해몽(解夢) | 꿈을 해석하는 것 |
| 몽점(夢占) | 꿈을 근거로 미래사를 점치는 것 |

> 출처: 한국민족문화대백과사전 "꿈" 항목 (encykorea.aks.ac.kr/Article/E0011278)

### 1.2 삼국시대 — 『삼국유사』 원성왕 사례

『삼국유사』 권2에 원성왕이 "복두를 벗고 흰 갓을 쓰고는 12현금을 들고 천관사 우물 속으로 들어가는 꿈"을 꾸었으나, 여삼 아찬이 긍정적으로 해석한 사례가 기록되어 있다.

→ 한국 해몽 전통은 최소 삼국시대까지 거슬러 올라가는 것이 1차 사료로 확인된다.

### 1.3 조선 — 『동의보감』 내경편 권02 "夢"

허준 『동의보감』(1613) 내경편 권02에는 꿈을 다음 항목으로 체계 분류한다:

| 한문 표제 | 내용 |
|-----------|------|
| 魂魄爲夢 | 혼백이 꿈을 만드는 기제 |
| 淫邪發夢 | 사기(邪氣)로 인한 꿈의 발생 |
| 五藏虛實爲夢 | 오장의 허실에 따른 꿈의 분류 |
| 陽氣之出入爲寤寐 | 양기의 출입으로 인한 수면/각성 |
| 昏沈多睡 | 혼침과 과다수면 |
| 虛煩不睡 | 허번으로 인한 불면 |

→ 동의보감은 꿈을 *민속 해몽*이 아닌 *한의학적 진단 지표*로 다룬다. 꿈을 신체 기능 상태의 반영으로 본 점이 특징이다.

> 출처: 위키문헌 『동의보감/내경편』 (ko.wikisource.org), 한국한의학연구원 미디어클래식스 (mediclassics.kr/books/8)

### 1.4 조선 후기 — 꿈이 기록된 주요 문헌

한국민족문화대백과사전이 한국 꿈 기록의 주요 문헌으로 인용하는 것:

- 『삼국유사』 (일연, 13세기)
- 『용재총화(慵齋叢話)』 (성현, 15세기 말)
- 『금오신화(金鰲新話)』 (김시습, 15세기)
- 『구운몽(九雲夢)』 (김만중, 1689)
- 『동야휘집(東野彙輯)』 (이원명, 1869)

> 주의: "조선 후기 민속해몽서·꿈풀이책 류"의 구체 표준 서명은 학술 1차 출처에서 단일 정본을 확인하기 어렵다. 위 문헌들은 *꿈을 다룬 기록*을 담은 문학·필기류이며, 시중에서 "민속해몽서"로 통칭되는 것과 직접 일치하지 않을 수 있다.

### 1.5 현대 시중 꿈사전류 — 학술 인용 부적합

- 시중 꿈해몽 사전(예: 『꿈풀이 큰사전』류), 점술 사이트(사주포럼·나테 운세 등), 인터넷 꿈해몽 페이지는 *출처 불명* 또는 *상업적 편집*이 다수다.
- 동일 상징에 대해 출판물마다 해석이 일관되지 않다.
- 학술 인용으로 사용하지 않는다.

## 2. 주요 상징 분류 (민속학적 보고)

> 아래 분류는 한국민족문화대백과사전·한국민속대백과사전이 *민속학적으로 정리한* 자료다. 검증된 과학적 인과관계가 아님을 다시 강조한다.

### 2.1 자연

| 상징 | 민속학적 풀이 |
|------|---------------|
| 해·달이 몸에 비춤 | 길몽 (한국민족문화대백과사전 인용) |
| 산이 무너짐 | 흉몽 (한국민족문화대백과사전 인용) |
| 물·바다·불 | 출처별로 길/흉 해석이 갈림 → 단정 금지 |

### 2.2 동물

| 상징 | 민속학적 풀이 (한국민속대백과사전·태몽 항목 등) |
|------|---------------|
| 용(龍) | 태몽 중 가장 상서로운 존재. 큰 인물 출산 상징으로 *전해진다* |
| 뱀·구렁이 | 태몽 시 부귀·권세 상징으로 *풀이되기도 한다* |
| 호랑이 | 권위·강한 힘 상징, 리더십 있는 인물 출산으로 *해석되기도 한다* |
| 돼지 | 한국 민속에서 재물 상징으로 *전해진다* |
| 새 | 자유·소식 등으로 *풀이되기도 한다* |

### 2.3 행위·사건

| 상징 | 민속학적 풀이 |
|------|---------------|
| 죽음 | 반몽(反夢) 해석으로 길몽으로 *풀이되는 경우가 있다* |
| 임신 | 태몽 범주 — 본 스킬 4장 참조 |
| 시험·결혼·돈 | 시중 사전에 풀이가 있으나 학술 1차 출처에서 단일 정설을 확인하기 어렵다 → 단정 금지 |
| 이가 빠짐·거울이 깨짐 | 흉몽 (한국민족문화대백과사전 인용) |

### 2.4 인물

| 상징 | 민속학적 풀이 |
|------|---------------|
| 조상·고인 | 한국 민속신앙에서 메시지 전달자로 *여겨지기도 한다* |
| 낯선 이 | 출처별 해석이 분기 → 단정 금지 |

## 3. 길몽 vs 흉몽 이분법 — 민속학적 배경과 한계

### 3.1 민속학적 배경

한국민족문화대백과사전은 길몽/흉몽을 다음과 같이 분류한 사례를 *민속 자료로* 보고한다:

- **길몽**: 용이 나타남, 해와 달이 몸에 비춤, 목욕
- **흉몽**: 이가 빠짐, 거울이 깨짐, 산이 무너짐

### 3.2 현대적 한계

1. 같은 상징이 출처별로 길/흉 반대로 해석되는 경우가 흔하다(반몽 해석).
2. 한국 민속의 길흉 이분법은 *해석자·시대·지역*에 따라 달라진 문화적 산물이며, 보편 예측 모델이 아니다.
3. "당신 꿈은 흉몽입니다" 같은 단정은 학술적 근거가 없다.

## 4. 태몽 — 한국 특유의 출산 관련 꿈 전통

### 4.1 정의

태몽(胎夢)은 *잉태할 조짐을 보여 준다고 믿는 꿈*이며, 한국 민속 생활 습속에서 잉태의 사실을 예측하는 방법으로 꿈을 풀이한 것이다.

> 출처: 한국민속대백과사전 "태몽" 항목 (folkency.nfm.go.kr/topic/detail/492), 한국민족문화대백과사전 "태몽" 항목 (encykorea.aks.ac.kr/Article/E0058917)

### 4.2 문화기술지(ethnography) 위치

- 태몽은 한국 민속 신앙 중 *현재까지 현장 조사로 확인 가능한 드문 사례*다.
- 서울대학교 간호학과 등에서 한국 여성 임신·출산 관련 focused ethnography 연구가 수행됨 (s-space.snu.ac.kr).
- 한국인의 태몽 유형·변천 연구가 학술지에 다수 발표됨 (scienceon.kisti.re.kr).

### 4.3 주요 상징 (민속학적 보고)

- 동물: 호랑이, 황룡, 돼지, 구렁이, 잉어, 자라, 수탉, 큰우렁, 물고기
- 식물: 복숭아 등 과실, 꽃
- 자연물: 해·달·별

### 4.4 적용 시 주의

- 태몽 해석을 "성별 예측"으로 단정하는 것은 *근거 없다*. 학술 연구도 성별 예측력은 통계적으로 입증되지 않았음을 보고한다.
- "이 꿈은 당신의 아이가 ~할 운명임을 의미합니다" 같은 단정 금지.

## 6. 앱 적용 시 톤 가이드 — 필수 준수

### 6.1 금지 표현

❌ "당신 꿈은 길몽입니다"
❌ "이번 주에 좋은 일이 생길 것입니다"
❌ "이 꿈은 임신 신호입니다"
❌ "돼지꿈 → 복권 사세요"

### 6.2 권장 표현 (hedging 톤)

✅ "민속학적으로는 …라고 풀이되기도 합니다"
✅ "한국 전통에서는 이 상징을 …로 해석한 기록이 있습니다 (출처: 한국민족문화대백과사전)"
✅ "한국민속대백과사전에 따르면 이런 꿈은 …로 전해져 왔습니다. 다만 이는 검증된 과학적 사실이 아닙니다"
✅ "꿈 해몽은 민속학적 자료이며, 단정적인 미래 예측으로 받아들이지 않는 것이 좋습니다"

### 6.3 응답 템플릿

```
{사용자 꿈 내용}을 한국 민속학 자료에서 찾아보면,
{한국민족문화대백과사전·한국민속대백과사전 인용}에서는
"{민속학적 풀이}"로 전해지기도 합니다.

다만 이는 검증된 과학적 사실이 아닌 민속학적 보고이므로,
단정적인 미래 예측으로 받아들이기보다는 한국 전통 문화의
일부로 참고하시기 바랍니다.

출처: {URL}
```

### 6.4 응답 시 필수 포함

1. 출처 명시 (1차 학술 자료 URL)
2. "민속학적 보고일 뿐"임을 매 응답마다 환기
3. 단정 표현 회피 (hedging 어미)
4. 사용자가 부정적 해석에 불안해할 가능성 대비 — *흉몽 해석 시 안심 문구 동봉*

## 8. 검증 한계 (UNVERIFIED 항목)

> 본 스킬에서 *학술 1차 출처로 단일 정본 확인이 어려운* 항목:

- 조선 후기 "민속해몽서·꿈풀이책 류"의 구체 표준 서명 — 한국민족문화대백과사전에 *꿈을 다룬 문헌* 명단은 있으나, "민속해몽서"라는 통칭에 정확히 대응하는 표준 서지 정보는 추가 학술 조사가 필요하다.
- 자연물(물·불·산·바다) 길흉 해석의 단일 정설 — 출처별 해석이 분기되므로 본 스킬에서는 한 가지로 단정하지 않는다.
- 행위(시험·결혼·돈) 꿈의 학술적 정설 — 시중 사전 풀이는 많으나 학술 출처가 약하다.
