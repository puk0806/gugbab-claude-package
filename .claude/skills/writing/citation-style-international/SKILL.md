# 국제 학술 인용 스타일 비교 + 인용관리 도구

> 소스: APA Style(https://apastyle.apa.org/), MLA Style Center(https://style.mla.org/), Chicago Manual of Style(https://www.chicagomanualofstyle.org/), Zotero(https://www.zotero.org/), Citation Style Language(https://citationstyles.org/)
> 검증일: 2026-05-03
> 대상: 도덕윤리교육 전공 대학원생 (영문 학술지 투고·국제 학회 발표)
> 한국 KCI 등재지 형식은 별도 스킬 참조 → `writing/academic-paper-structure-humanities`

---

## 0. 스타일 선택 빠른 가이드

| 분야 | 권장 스타일 | 이유 |
|------|-------------|------|
| 윤리학·교육학 (사회과학적 접근) | **APA 7판** | 사회과학 표준, author-date 시스템 |
| 도덕철학·윤리 사상사 (인문학적 접근) | **Chicago 17/18판 NB** | 각주 활용, 인문학 전통 |
| 문학·서양고전학 | **MLA 9판** | 인문학·문학 연구 표준 |
| 영국·호주·유럽 일부 학술지 | **Harvard** | 단일 표준 없음, 학술지별 변형 |
| 의학·생명과학 | **Vancouver** | 도덕윤리교육과 보통 무관 |

> 핵심 원칙: 투고할 학술지의 author guidelines를 먼저 확인하고, 그 학술지가 명시한 스타일을 따른다. 학회 발표 자료도 학회 측 템플릿 우선.

## 1. APA 7판 (American Psychological Association) — 사회과학 표준

> 출처: https://apastyle.apa.org/style-grammar-guidelines/citations
> Publication Manual 7th edition (2020)

### 1-1. 본문 내 인용 (In-text Citation) — Author-Date 시스템

**괄호형(Parenthetical)과 서술형(Narrative)**

| 저자 수 | 괄호형 | 서술형 |
|--------|-------|-------|
| 1명 | (Luna, 2020) | Luna (2020) |
| 2명 | (Salas & D'Agostino, 2020) | Salas and D'Agostino (2020) |
| 3명 이상 | (Martin et al., 2020) | Martin et al. (2020) |

- 괄호형에서는 `&`(앰퍼샌드), 서술형에서는 `and`를 철자로 쓴다.
- 직접 인용 시 페이지 번호 추가: `(Luna, 2020, p. 45)` 또는 `(Luna, 2020, pp. 45–47)`.

### 1-2. 참고문헌 (References)

- 첫 줄 정렬 + **hanging indent** (둘째 줄부터 들여쓰기)
- 알파벳 순서로 정렬
- 학술지명·도서명: *이탤릭*

**학술지 논문**
```
Author, A. A., & Author, B. B. (Year). Title of article. Journal Name, Volume(Issue), pp–pp. https://doi.org/xxxxx
```

**단행본**
```
Author, A. A. (Year). Title of book. Publisher.
```

**편집된 책의 챕터**
```
Author, A. A. (Year). Title of chapter. In E. Editor (Ed.), Title of book (pp. xx–xx). Publisher.
```

**웹사이트 자료**
```
Author, A. A. (Year, Month Day). Title of webpage. Site Name. URL
```

### 1-3. 번역서 인용 (APA 7판)

**본문 내 인용**: 원저 연도 / 번역 연도 슬래시 표기
```
(Aristotle, ca. 350 B.C.E./2009)
(Piaget, 1970/1972)
```

**참고문헌 형식**
```
Author, A. A. (Translation Year). Title of book (T. Translator, Trans.). Publisher. (Original work published Year)
```

예시:
```
Aristotle. (2009). Nicomachean ethics (D. Ross, Trans.; L. Brown, Rev. ed.).
    Oxford University Press. (Original work published ca. 350 B.C.E.)
```

> 주의: 위 Aristotle 예시의 reviser(L. Brown) 표기 형식은 APA Publication Manual 본문 예시에는 명시되지 않는다. APA 공식 블로그의 번역서 예시(Piaget, Coelho 등)를 따른 응용이며, 학술지 제출 시 편집자 가이드를 확인할 것.

## 2. MLA 9판 (Modern Language Association) — 인문학·문학 표준

> 출처: https://style.mla.org/
> MLA Handbook 9th edition (2021)

### 2-1. 본문 내 인용 — Author-Page 시스템 (연도 없음)

```
(저자성 페이지)
```

| 상황 | 형식 | 예시 |
|------|------|------|
| 1명 저자 | (Author Page) | (Baron 194) |
| 2명 저자 | (Author1 and Author2 Page) | (Dorris and Erdrich 23) |
| 3명 이상 | (Author1 et al. Page) | (Franck et al. 327) |
| 본문에 저자 언급 | (Page) | "..." (3) |
| 페이지 없음 (웹) | (Author) | (Baron) |

### 2-2. Works Cited — 9개 핵심 요소(Core Elements)

MLA 9판은 자료 유형별 개별 형식이 아닌 **9개 핵심 요소 템플릿**을 사용한다:

```
(1) Author.
(2) "Title of Source."
(3) Title of Container,
(4) Other contributors,
(5) Version,
(6) Number,
(7) Publisher,
(8) Publication date,
(9) Location.
```

- 각 요소 끝에 마침표 또는 쉼표 (마지막은 마침표).
- 해당 없는 요소는 건너뛴다 (단, **Title은 절대 생략 금지**).
- hanging indent 적용.

**학술지 논문 예시**
```
Author Last, First. "Title of Article." Journal Name, vol. X, no. Y, Year, pp. xx–xx.
```

**단행본 예시**
```
Author Last, First. Title of Book. Publisher, Year.
```

### 2-3. 번역서 인용 (MLA 9판)

**저자 중심 (저자에 초점)**:
```
Homer. The Iliad. Translated by Robert Fagles, Penguin, 1990.
```

**번역자 중심 (번역에 초점)**:
```
Reck, Michael, translator. The Iliad. By Homer, HarperCollins, 1994.
```

> 어떤 형식을 쓸지는 인용에서 어디에 초점을 두는가에 따라 결정된다 (저자의 사상 vs 번역의 특성).

## 3. Chicago 17판 (Chicago Manual of Style)

> 출처: https://www.chicagomanualofstyle.org/tools_citationguide.html
> Chicago Manual of Style 17th edition (2017)

> 주의: Chicago Manual of Style 18판이 2024년 9월에 출시되었다. 18판 주요 변경사항은 인용 출판지(publication location) 생략 권장, AI 생성 텍스트·이미지 인용 양식 추가, 한국어 표기 가이드 확장 등이다. 학술지 투고 시 18판 사용 여부를 편집자에게 확인할 것.

Chicago는 두 가지 시스템이 공존한다:
- **Notes-Bibliography (NB)**: 인문학(문학·역사·예술·철학) 전통
- **Author-Date**: 사회과학용

### 3-1. Notes-Bibliography (NB) 스타일

#### 학술지 논문

**First Full Note (첫 전체 각주)**
```
1. Hyeyoung Kwon, "Inclusion Work: Children of Immigrants Claiming
   Membership in Everyday Life," American Journal of Sociology 127,
   no. 6 (2022): 1842–43, https://doi.org/10.1086/720277.
```

**Shortened Note (이후 약식 각주)**
```
2. Kwon, "Inclusion Work," 1851.
```

**Bibliography (참고문헌 항목)**
```
Kwon, Hyeyoung. "Inclusion Work: Children of Immigrants Claiming
    Membership in Everyday Life." American Journal of Sociology 127,
    no. 6 (2022): 1818–59. https://doi.org/10.1086/720277.
```

#### 단행본

**First Full Note**
```
1. Charles Yu, Interior Chinatown (Pantheon Books, 2020), 45.
```

**Shortened Note**
```
2. Yu, Interior Chinatown, 48.
```

**Bibliography**
```
Yu, Charles. Interior Chinatown. Pantheon Books, 2020.
```

#### Ibid. 사용에 관한 17판 변경사항

> CMOS 17판부터 ibid.(ibidem) 사용을 권장하지 않는다. 전자 출판물에서 한 번에 하나의 각주만 표시되는 환경 때문에 ibid.가 독자에게 혼란을 준다. 대신 shortened note(약식 각주) 사용을 권장한다.

### 3-2. Author-Date 스타일

#### 학술지 논문

**Reference List 항목**
```
Dittmar, Emily L., and Douglas W. Schemske. 2023. "Temporal Variation in
    Selection Influences Microgeographic Local Adaptation." American
    Naturalist 202 (4): 471–85.
```

**본문 내 인용**: `(Dittmar and Schemske 2023, 480)`

#### 단행본

**Reference List 항목**
```
Binder, Amy J., and Jeffrey L. Kidder. 2022. The Channels of Student
    Activism. University of Chicago Press.
```

**본문 내 인용**: `(Binder and Kidder 2022, 117–18)`

### 3-3. NB와 Author-Date 핵심 차이

| 항목 | Notes-Bibliography | Author-Date |
|------|--------------------|--|
| 인용 위치 | 각주(footnote/endnote) | 본문 괄호 |
| 출판 연도 위치 | 출판사·장소 뒤 | 저자명 직후 |
| 제목 표기 | 따옴표 또는 이탤릭 | 따옴표 또는 이탤릭 |
| 사용 분야 | 문학·역사·예술·철학 | 사회과학 |

## 4. Harvard 스타일

- 본문 내 인용: `(Author Year)` 또는 `(Author Year, p. XX)`
- References: 알파벳순 정렬

> 주의: Harvard는 단일 공식 표준이 없다. 기관·학술지별 변형이 다양하므로 (예: Harvard Anglia, Harvard AGSM, Harvard Cite Them Right), 투고처가 Harvard를 요구하면 반드시 그 기관의 세부 가이드를 확인한다.

---

## 5. Vancouver 스타일 — 의학 표준 (도덕윤리교육에서는 보통 무관)

- 본문 내 인용: 숫자 위첨자 또는 괄호 `[1]`, `[2,3]`
- References: 본문 등장 순서 (알파벳순 아님)

> 도덕윤리교육 학위 논문·학회 발표에서는 거의 사용하지 않는다. 의학 윤리(bioethics) 학술지 일부에서 요구할 수 있어 참고만 한다.

## 6. 고대 텍스트 인용 — 모든 스타일 공통 원칙

### 6-1. 표준 번호 체계

| 저자 | 번호 체계 | 형식 | 의미 |
|------|-----------|------|------|
| 아리스토텔레스 | **Bekker 번호** | `1098a16` | Bekker 판 페이지·단(a/b)·줄 번호 |
| 플라톤 | **Stephanus 번호** | `352b` | Stephanus 판 페이지·단락(a–e) |

> 이 번호 체계는 어떤 번역본·판본을 사용해도 동일한 위치를 가리킬 수 있는 학계 합의다. 페이지 번호로 인용하면 안 된다.

### 6-2. APA 7판 — 고전 인용

> 출처: https://apastyle.apa.org/blog/citing-classical-religious-works

> APA는 고대 그리스 텍스트를 참고문헌에 반드시 등록할 필요가 없다고 명시하지만, 본문에서 사용한 판본을 명확히 밝히면 된다. 다만 학위 논문에서는 일반적으로 reference list에 포함하는 것을 권장한다.

**본문 내 인용 형식**:
```
"Being happy takes a complete lifetime" (Aristotle, Nicomachean Ethics,
    1.7, 1098a16).

(Plato, Phaedrus, 274c)
```

- `1.7`은 책·장 번호, `1098a16`이 Bekker 페이지·단·줄.
- 페이지 번호 대신 **canonically numbered parts**(책, 장, 절, 행, 칸토 등)를 사용한다.

### 6-3. MLA 9판 — 고전 인용

```
(Aristotle, Nicomachean Ethics, 1094a1-2)
(Aristotle, Metaphysics, 1.9, 991b9-20)
(Plato, Phaedrus 274c)
```

**Works Cited 예시**:
```
Plato. Republic. Translated by G. M. A. Grube, revised by C. D. C. Reeve,
    Hackett, 1992.
```

### 6-4. Chicago 17판 — 고전 인용

각주에 표준 약어 사용 권장:

| 약어 | 원제 |
|------|------|
| NE | Nicomachean Ethics |
| EE | Eudemian Ethics |
| Pol. | Politics |
| Resp. | Republic (Plato) |
| Phdr. | Phaedrus (Plato) |

각주 예시:
```
1. Aristotle, NE 1.7, 1098a16.
2. Plato, Resp. 514a–520a.
```

> 주의: 학술지마다 약어 표기 정책이 다르다. *Classical Quarterly*, *Phronesis*, *Journal of the History of Philosophy* 등 주요 철학·고전학 학술지 가이드를 우선 따른다.

### 6-5. 번호 표기 형식 비교

도덕윤리교육 학위 논문에서 자주 쓰이는 *Nicomachean Ethics* 7권 3장의 한 구절을 예로 들면:

| 스타일 | 표기 |
|--------|------|
| 풀 형식 | NE VII.3, 1147a24-b5 |
| 약식 (책/장 생략) | NE 1147a24-b5 |
| APA 7 본문 | (Aristotle, *Nicomachean Ethics*, 7.3, 1147a24–b5) |
| MLA 9 본문 | (Aristotle, *Nicomachean Ethics*, 1147a24-b5) |
| Chicago NB 각주 | Aristotle, NE 7.3, 1147a24–b5. |

## 7. 인용관리 도구 (Reference Management Tools)

### 7-1. Zotero (가장 권장) — 무료·오픈소스

> 출처: https://www.zotero.org/
> 최신 버전: Zotero 7 (2024년 8월 9일 출시)

**핵심 기능**:
- 브라우저 확장(Zotero Connector)으로 학술 DB·웹사이트의 메타데이터·PDF 자동 수집
- Microsoft Word, LibreOffice, Google Docs 통합 플러그인 (Zotero 7부터 자동 설치)
- EPUB 및 웹페이지 스냅샷 주석 (Zotero 7 신규 기능)
- 다크 모드, Apple Silicon 네이티브 지원
- **Group Library**: 공동 연구·학회 발표팀 협업
- 클라우드 동기화 (개인 기본 300MB 무료)

**워드프로세서 통합**:
- Word/LibreOffice: Zotero 7 설치 시 자동으로 플러그인 추가
- Google Docs: 브라우저 확장으로 통합
- 본문 인용 삽입 → 참고문헌 자동 갱신

**추천 이유**:
- 단일 회사에 종속되지 않는 오픈소스 (장기 사용 안전)
- 학술 표준 CSL 완전 지원
- 모든 OS 지원 (Windows/macOS/Linux)

### 7-2. Mendeley (Reference Manager)

> 출처: https://www.mendeley.com/
> Elsevier 운영

**현황**:
- Mendeley Desktop은 개발 중단, 신규 사용자는 **Mendeley Reference Manager** 사용
- Mendeley Cite (Word add-in)로 인용 삽입
- PDF 관리·하이라이트·노트북 기능
- AI 기반 논문 비교·탐색 기능 추가 중

**적합한 경우**:
- Elsevier 학술지(ScienceDirect)에 자주 투고하는 경우
- PDF 기반 작업 중심 워크플로우

> 주의: Elsevier 정책 변경 위험 있음 (장기 보존성은 Zotero보다 떨어진다).

### 7-3. EndNote — 유료, 한국 일부 대학 라이센스

> 출처: https://endnote.com/
> Clarivate 운영. 최신 버전: EndNote 2025

**현황**:
- EndNote 21 → EndNote 2025로 업데이트
- 라이브러리 공유 최대 1,000명 (대규모 팀 연구 적합)
- 활동 피드(Activity Feed) 기능
- Word 통합, EndNote Click(브라우저 확장)

**적합한 경우**:
- 소속 대학·기관이 라이센스를 제공하는 경우
- 대규모 공동 연구 (수백 명 단위 라이브러리 공유)

### 7-4. BibTeX — LaTeX 사용자

- LaTeX 문서에서 `\cite{key}`로 인용 삽입
- `.bib` 파일에 서지 정보 저장
- BibLaTeX(현대화 버전) 권장
- Zotero에서 BibTeX 파일 export 가능 (LaTeX 사용자도 Zotero를 메타데이터 관리에 활용 가능)

### 7-5. 도구별 스타일 변경법

| 도구 | 스타일 변경 방법 |
|------|-----------------|
| Zotero 7 (2024.8 이후) | Edit → **Settings** → Cite → Styles → "Get additional styles…" (구버전은 Edit → **Preferences**, 기능 동일) — CSL 파일 직접 설치는 "+" 버튼으로 로컬 .csl 파일 추가 가능 |
| Mendeley Reference Manager | Mendeley Cite (Word) → Citation Style |
| EndNote | Tools → Output Styles → Open Style Manager |
| BibTeX | `\bibliographystyle{}` 명령 또는 BibLaTeX `style=` 옵션 |

## 8. CSL (Citation Style Language)

> 출처: https://citationstyles.org/, 공식 저장소: https://www.zotero.org/styles

**CSL 개요**:
- Zotero, Mendeley, Papers, RefWorks 등이 공통으로 사용하는 **XML 기반 인용 양식 언어**
- 10,000개 이상의 학술지·기관 스타일 무료 제공
- Creative Commons BY-SA 라이센스
- 현재 표준 버전: CSL 1.0.2

**활용 절차**:
1. https://www.zotero.org/styles 접속
2. 검색창에 학술지명 또는 학회명 입력 (예: "Educational Researcher", "Journal of Moral Education")
3. 해당 스타일을 클릭하면 Zotero에 자동 설치
4. Mendeley도 같은 CSL 파일을 사용하므로 호환

**한국 KCI 등재지 검색**:
- 일부 한국 학술지(특히 인문·사회 분야)는 Zotero Style Repository에 등록되어 있다
- 검색해도 안 나오면 공식 저장소(github.com/citation-style-language/styles)에 요청 가능
- 한국어 인문학 연구용 CSL은 별도 공개되어 있다 (예: github.com/go00ood/zotero-style-korean)

> 주의: Zotero Style Repository에 KCI 등재지 전체가 포함되어 있지는 않다. 투고 학술지의 양식이 없으면 가장 가까운 표준(Chicago NB 또는 APA)을 변형해 사용하거나 수동 정리한다.

## 9. 흔한 실수 (Common Mistakes)

| 실수 | 잘못된 예 | 올바른 예 |
|------|-----------|-----------|
| **스타일 혼용** | 한 논문에 APA + Chicago 동시 사용 | 학술지 가이드의 단일 스타일 일관 적용 |
| **한글 폰트 혼용** | 영문 인용에 본문체(맑은 고딕) 한글 폰트 적용 | 영문은 Times New Roman·Arial 등 영문 전용 |
| **번역서 원저자 누락** | "Ross, D. (2009). Nicomachean ethics." | Aristotle을 저자로, Ross를 번역자로 |
| **고전 페이지 번호** | (Aristotle, NE, p. 156) | (Aristotle, NE, 1098a16) — Bekker 번호 |
| **URL 미명시 (전자 자료)** | "Smith, 2023. Educational Researcher." | DOI 또는 안정적 URL 포함 |
| **Ibid. 남용 (Chicago 17·18판)** | 연속된 각주에 Ibid., Ibid. | shortened note 사용 (Yu, *Interior Chinatown*, 48.) |
| **MLA에 연도 표기** | (Smith 2020, 45) | (Smith 45) — MLA는 연도 없음 |
| **APA 페이지 번호 형식** | (Smith, 2020, page 45) | (Smith, 2020, p. 45) |
| **et al. 적용 기준 혼동** | APA에서 2명 저자에 et al. 적용 | APA는 3명 이상부터 et al. |
| **하위 디렉토리 자동 생성된 CSL 미검증** | 다운받은 CSL을 검토 없이 사용 | 미리보기로 출력 확인 후 사용 |

## 10. 도덕윤리교육 분야 실전 권장 워크플로우

1. **투고 전**: 대상 학술지의 Author Guidelines 확인 → 스타일 결정 (예: *Journal of Moral Education*은 APA 7판 사용)
2. **자료 수집**: Zotero Connector로 PhilPapers, JSTOR, ERIC, Google Scholar 등에서 메타데이터 자동 수집
3. **고전 텍스트**: Aristotle/Plato 등은 직접 입력 — Bekker/Stephanus 번호로 본문 인용
4. **번역서**: 원저자 + 번역자 + 원저 발행 연도 모두 입력
5. **본문 작성**: Word + Zotero 플러그인으로 인용 삽입 → 참고문헌 자동 생성
6. **최종 검토**: 자동 생성된 참고문헌이 학술지 가이드와 정확히 일치하는지 수동 확인

## 참고: 한국 KCI 등재지 형식

영문 학술지·국제 학회가 아닌 **한국어 학술지(KCI)** 투고 시에는 별도 가이드를 따른다:

→ 별도 스킬 참조: `writing/academic-paper-structure-humanities`

KCI 등재지(예: 「도덕윤리과교육연구」, 「윤리연구」)는 자체 양식을 가지고 있으며, 본 스킬의 국제 스타일과 혼용해서는 안 된다.
