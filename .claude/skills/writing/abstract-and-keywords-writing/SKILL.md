# 학술 논문 초록·키워드 작성 + 검색 노출 최적화

> 소스: APA 7판 Abstract & Keywords Guide(https://apastyle.apa.org/instructional-aids/abstract-keywords-guide.pdf), Springer Nature Titles·Abstracts·Keywords(https://www.springernature.com/gp/authors/campaigns/writing-a-manuscript/titles-abstracts-keywords), Elsevier Researcher Academy(https://researcheracademy.elsevier.com/writing-research/fundamentals-manuscript-preparation/write-abstract-improve-article), PhilPapers Categorization(https://philpapers.org/help/categorization.html), UKSG Insights ASEO(https://insights.uksg.org/articles/10.1629/uksg.534), Purdue OWL Abstracts(https://owl.purdue.edu/owl/graduate_writing/graduate_writing_genres/graduate_writing_genres_abstracts_new.html), Editage Insights 한국어 시제·인칭(https://www.editage.co.kr/insights/which-tense-should-be-used-in-the-abstract-of-a-paper), KCI(https://www.kci.go.kr/)
> 검증일: 2026-05-03

## 1. 초록(Abstract) 표준 구조

초록은 학술지·분야에 따라 두 가지 형태로 나뉜다. **반드시 투고 학술지의 투고규정을 먼저 확인**한 뒤 형식을 선택한다.

### 1-1. 구조화 초록 (Structured Abstract) — 의학·실증 표준

명시적 소제목으로 분리되며 Elsevier·Springer 의학·생명과학 저널과 일부 사회과학 저널에서 채택한다.

| 소제목 | 1-2문장으로 작성할 내용 |
|--------|-------------------------|
| Background / 배경 | 연구가 다루는 문제와 그 중요성 |
| Objective / 목적 | 연구 질문 또는 가설 |
| Methods / 방법 | 연구 설계, 표본, 분석 기법 |
| Results / 결과 | 핵심 수치·발견 |
| Conclusions / 결론 | 함의, 기여, 한계 |

> 출처: Elsevier 구조화 초록 표준 (최대 250단어)

### 1-2. 비구조화 초록 (Unstructured Abstract) — 인문학·교육학 표준

한 문단으로 작성하며 소제목이 없다. 인문학·철학·교육학·MLA 분야 다수 저널에서 채택한다.

**인문학 초록 5요소 패턴:**

| 문장 | 역할 |
|------|------|
| 1문장 | **문제 제기** — 왜 이 주제가 중요한가, 어떤 문제 상황이 있는가 |
| 2문장 | **본 연구의 위치** — 선행 연구의 한계 또는 갭 |
| 3-4문장 | **핵심 논증·접근** — 본 연구가 채택한 관점·방법, 주요 논지 |
| 5문장 | **결론·기여** — 본 연구가 학계·실천에 어떤 의미를 갖는가 |

> 인문학 학술지는 MLA처럼 초록을 요구하지 않는 경우도 있으므로 **투고규정 우선 확인** 필수.

## 2. 분량 표준 (학술지별 차이)

> 주의: 아래 수치는 **일반 권장값**이며, 항상 투고 학술지의 투고규정이 우선한다.

| 학술지·표준 | 권장 분량 |
|-------------|-----------|
| APA 7판 | 150-250 단어 (단일 문단, 들여쓰기 없음) |
| MLA (인문학) | 100-250 단어 (정보형 150-250 / 서술형 100 내외) |
| Springer Nature | 200 단어 이내, 소제목·수식·인용 금지 |
| Elsevier (구조화) | 250 단어 이내 |
| Elsevier (비구조화) | 150 단어 이내인 경우도 있음 |
| KCI 등재지 한국어 | 일반적으로 **600-1000자** 범위, 학술지마다 다름 |
| KCI 등재지 영문 초록 | 150-250 단어 (한국어 초록과 별도 요구되는 경우 다수) |

> 주의: KCI 등재지의 한국어 초록 분량은 **학술지마다 편차가 크다**. "도덕윤리과교육", "초등도덕교육" 등 개별 학술지 투고규정을 직접 확인해야 한다. "800자 내외"는 흔한 기준 중 하나일 뿐 절대 기준이 아니다.

## 3. 인문학·도덕교육 초록 작성 패턴 (예시 골격)

```
[1문장 — 문제 제기]
도덕적 의지의 약함, 즉 아크라시아(akrasia)는 윤리학에서 오랫동안
다뤄졌으나, 도덕과 교육 현장에서 학습자의 행위 동기를 설명하는
틀로는 충분히 활용되지 못했다.

[2문장 — 선행연구 갭]
선행 연구는 아크라시아를 주로 인지·의지 차원에서 다루어 왔으며,
2022 개정 도덕과 교육과정의 핵심역량과 연결한 논의는 드물다.

[3-4문장 — 핵심 논증·접근]
본 연구는 아리스토텔레스 『니코마코스 윤리학』 제7권의 아크라시아
논의를 재구성하고, 이를 도덕적 자기조절 역량 함양과 연결한다.
이를 위해 텍스트 분석과 교육과정 문서 분석을 병행한다.

[5문장 — 결론·기여]
본 연구는 도덕과 수업에서 아크라시아 개념이 학습자의 의지적 실천을
설명·지도하는 교육적 자원으로 활용될 수 있음을 보이며,
2022 개정 교육과정 운영에 시사점을 제공한다.
```

> 한국어 학술 글쓰기에서는 "본 연구는~", "본고는~"처럼 3인칭으로 자기를 지칭하는 것이 일반적이다. 다만 철학 논문에서는 "필자" 또는 "나"를 사용하기도 한다. 출처: Enago Korea, Editage Insights.

## 4. 흔한 실수 (반드시 회피)

| 실수 | 올바른 대안 |
|------|-------------|
| 초록에 참고문헌 인용 포함 | 본문에서 인용. Springer 공식: "abstract는 인용을 포함하지 않는다" |
| 약어를 정의 없이 사용 | 첫 등장 시 풀어 쓰고 괄호로 약어 제시 (단, 매우 보편적인 약어 제외) |
| 1인칭("나") vs 3인칭("본 연구") 혼용 | 학술지·분야 관행에 따라 **하나로 통일** |
| 결론·기여 없는 단순 요약 | "본 연구의 기여는 ~이다"를 마지막 문장에 명시 |
| 미래 시제("본 연구는 ~할 것이다") | 이미 완료된 연구는 현재·과거 시제 사용 |
| 소제목·수식·표·그림 삽입 | Springer/APA 공식: 초록은 단일 문단, 시각 요소 금지 |
| 학술지 제한 단어 수 초과 | 투고규정의 상한 엄수 |

### 시제 사용 규칙 (Editage Insights)

| 진술 유형 | 권장 시제 |
|-----------|-----------|
| 일반적 사실 | 현재 |
| 선행 연구 언급 | 과거 |
| 본 연구의 결과·관찰 | 과거 |
| 결론·해석 | 현재 |
| 본 논문 자체의 진술("본 연구는 ~을 다룬다") | 현재 |

## 5. 키워드(Keywords) 선정 전략

### 5-1. 개수

| 표준 | 권장 개수 |
|------|-----------|
| Springer Nature | 4-6개 |
| Elsevier | 학술지마다 다름 (대개 3-6개) |
| APA | 일반적으로 3-5개 |
| ASEO 권장 (UKSG Insights) | **3-7개** — 그 이상은 검색엔진이 spam으로 처리 |

### 5-2. 검색 노출 최적화 원칙 (ASEO)

1. **분야 표준 용어 우선**: 학계에서 통용되는 정확한 술어를 사용한다.
   - 예: "akrasia" + "weakness of will" + "moral motivation"
2. **너무 광범위한 용어 회피**: "윤리학", "교육", "철학" 같은 단어는 검색 결과가 너무 많아 본 논문이 묻힌다.
   - Springer 공식 예시: "carbon nanotubes, Kerr effect" (O) vs "molecule, optics" (X)
3. **동의어·이형 표기 포함**: 다른 검색어로도 발견되도록 한다.
   - 예: `akrasia` / `아크라시아` / `자제력 없음` / `의지박약`
4. **영문/한글 병기 권장**: 국제 색인 노출을 위해 영문 키워드를 함께 등록한다.
5. **약어보다 풀이 우선**: 첫 등장에서는 풀이된 형태를 키워드로 등록한다.
6. **MeSH·LCSH 활용** (해당 분야 한정): 의학(MeSH), 인문사회(LCSH) 통제 어휘 사용 시 데이터베이스 색인 정확도가 올라간다.

### 5-3. 키워드 배치 (검색 노출 가속)

ASEO 핵심 원칙: **제목 → 초록 첫 두 문장 → 키워드** 순으로 핵심 용어가 모두 등장해야 검색 가중치가 높아진다.

- 제목에 핵심 키워드 1-2개 포함
- 초록 첫 1-2 문장에 핵심 키워드가 자연스럽게 등장
- 초록 전체에서 핵심 키워드가 **3-5회** 자연스럽게 반복
- 키워드 필드에 동의어·이형 포함

> 출처: Springer Nature 공식 — "키워드는 indexer와 검색엔진이 관련 논문을 찾도록 돕는 도구"

## 6. KCI 검색 노출 최적화

KCI는 한국연구재단이 운영하는 국내 학술지 인용색인 시스템으로, **제목 + 키워드 + 초록**의 텍스트를 인덱싱한다.

### 6-1. 한국어·영문 키워드 동시 등록

대부분의 KCI 등재지는 한국어 키워드와 영문 키워드를 모두 요구한다. 두 언어 키워드가 정확히 1:1 대응되지 않아도 되며, 의미가 같다면 서로 다른 표기를 등록해 검색 노출을 넓힐 수 있다.

### 6-2. 동의어·이형 표기 활용

한국어 학술 용어는 외래어·한자어·번역어가 공존한다. 검색자가 어떤 표기로 검색할지 모르므로 키워드 필드와 초록 본문에 동시에 노출시킨다.

```
한글 키워드: 아크라시아, 자제력 없음, 도덕적 의지, 아리스토텔레스,
            도덕윤리과교육, 2022 개정 교육과정
영문 키워드: akrasia, weakness of will, moral motivation, Aristotle,
            Nicomachean Ethics, moral education
```

### 6-3. KCI 색인 작동 패턴 (실무 관찰)

- **제목**의 단어가 가장 높은 가중치를 갖는다.
- **저자 키워드** 필드는 두 번째 가중치.
- **초록 본문**에서 등장하는 빈도도 검색 노출에 영향을 준다.

> 주의: KCI는 공식적인 랭킹 알고리즘 문서를 공개하지 않는다. 위 패턴은 일반적인 색인 시스템 동작에 기반한 추론이며, 절대 기준이 아니다.

## 7. JSTOR · PhilPapers · Google Scholar 검색 노출

### 7-1. PhilPapers (철학 분야)

- 5단계 카테고리 시스템: Cluster → Area → Subarea → Topic → Subtopic.
- 모든 논문은 **1-3개의 leaf 카테고리**로 분류된다.
- 제출 시 abstract, categories, keywords, link를 모두 입력해야 발견 가능성이 높다.
- akrasia 논문이라면 `Value Theory > Ethics > Normative Ethics > Moral Psychology` 계열 leaf 카테고리에 등재.

> 출처: PhilPapers Categorization Project (philpapers.org/help/categorization.html)

### 7-2. JSTOR

- JSTOR는 출판사가 제공한 메타데이터(제목·초록·키워드)를 그대로 색인한다.
- 사용자가 수동으로 등재 카테고리를 바꿀 수는 없으므로 **투고 시점의 키워드 품질이 결정적**이다.

### 7-3. Google Scholar

- Google Scholar는 PDF의 텍스트와 메타데이터를 함께 인덱싱한다.
- 핵심 가중치 요소: 제목, 저자, 인용 수, 발행 학술지.
- **인용 수가 누적될수록 노출이 가속**된다 → 첫 1-2년의 동료 인용을 받기 위한 키워드 정확도가 중요.
- Google Scholar에 잘 노출되는 학술지·출판사를 선택하는 것 자체가 ASEO 전략이다.

> 출처: UKSG Insights — Increasing visibility and discoverability with academic SEO

## 8. akrasia 논문 키워드 예시

도덕윤리과교육 분야 논문 기준 (한·영 6개씩):

```
한글 키워드:
  아크라시아, 자제력 없음, 도덕적 의지,
  아리스토텔레스, 도덕윤리과교육, 2022 개정 교육과정

영문 키워드:
  akrasia, weakness of will, moral motivation,
  Aristotle, Nicomachean Ethics, moral education
```

**전략 분석:**
- "아크라시아"·"akrasia": 분야 표준 술어 (PhilPapers leaf 카테고리와 매칭)
- "자제력 없음"·"weakness of will": 가장 흔한 번역어 (검색자가 사용할 가능성 높음)
- "도덕적 의지"·"moral motivation": 인접 연구자 검색어
- "아리스토텔레스"·"Aristotle"·"Nicomachean Ethics": 사상가·텍스트 명 (PhilPapers, JSTOR 색인 강력)
- "도덕윤리과교육"·"moral education": 분야명 (KCI 등재지 노출 핵심)
- "2022 개정 교육과정": 시의성 키워드 (현직 교사·정책 연구자 검색)

## 9. 초록 작성 후 셀프 체크리스트

투고 전 다음 항목을 모두 점검한다.

- [ ] **이해 가능성**: 한 문단으로 읽었을 때 본 연구가 무엇을 했는지, 왜 했는지, 무엇을 발견했는지 명확한가
- [ ] **검색 키워드 반복**: 핵심 용어가 초록에서 3-5회 자연스럽게 등장하는가
- [ ] **첫 문장 핵심어**: 초록 첫 1-2 문장에 가장 중요한 키워드가 들어가는가
- [ ] **차별점·기여 명시**: "본 연구의 기여는 ~이다"가 마지막 문장에 있는가
- [ ] **분량 준수**: 학술지 투고규정의 단어/자수 상한을 지켰는가
- [ ] **인용 없음**: 초록 안에 참고문헌 인용(Smith, 2024 등)이 없는가
- [ ] **약어 처리**: 약어가 모두 풀이되었거나 매우 보편적인 약어인가
- [ ] **인칭 통일**: 1인칭("필자/나") 또는 3인칭("본 연구")이 일관되게 사용되는가
- [ ] **시제**: 일반 사실은 현재, 결과는 과거, 결론은 현재로 구분되는가
- [ ] **시각 요소 없음**: 표·그림·수식·소제목이 들어 있지 않은가 (구조화 초록 제외)
- [ ] **한·영 키워드 정합**: 한글·영문 키워드의 의미가 일치하고 동의어가 포함되었는가
- [ ] **키워드 개수**: 학술지 권장 범위(대개 3-7개) 내인가

## 10. AI 활용 시 윤리 (research-ethics-and-integrity 스킬과 연계)

> 본 절은 **연계 지침**이며 강제 규정의 출처는 각 학술지·출판사 정책이 우선한다.

| 작업 | 가이드 |
|------|--------|
| ChatGPT/Claude로 초록 **다듬기**(문법·문장 매끄럽게) | 일반적으로 허용. 학술지가 AI 활용 명시 의무를 요구하면 "감사의 글" 또는 "방법" 절에 사용 사실 기재 |
| AI에게 초록 **자동 생성** 요청 | 저자성(authorship) 위반 위험. **권장하지 않음** — 초록은 저자의 학술적 판단이 담긴 핵심 텍스트 |
| AI에게 키워드 **추천** 받기 | 허용. 다만 추천된 키워드를 분야 표준에 맞춰 저자가 검토·확정한다 |
| AI 사용 명시 | Elsevier·Springer·COPE 등 주요 출판사는 AI 사용 시 명시를 권고한다. 투고 전 학술지 정책 확인 필수 |

> 핵심 원칙: **AI는 도구이며 저자가 아니다.** 초록·키워드의 학술적 책임은 항상 인간 저자에게 있다.

## 부록: 학술지별 빠른 참조표

| 학술지/표준 | 초록 분량 | 키워드 개수 | 형식 |
|-------------|-----------|-------------|------|
| APA 7판 | 150-250 단어 | 3-5개 | 단일 문단, 들여쓰기 없음 |
| MLA (인문학) | 100-250 단어 | 학술지마다 다름 | 단일 문단 |
| Springer Nature | 200 단어 이내 | 4-6개 | 단일 문단, 인용·수식·소제목 금지 |
| Elsevier (구조화) | 250 단어 이내 | 학술지마다 다름 | Background/Methods/Results/Conclusion |
| Elsevier (비구조화) | 150 단어 내외 | 학술지마다 다름 | 단일 문단 |
| KCI 등재지 한국어 | 학술지마다 다름 (600-1000자 흔함) | 5-7개 흔함 | 단일 문단, 영문 초록 별도 요구 다수 |
