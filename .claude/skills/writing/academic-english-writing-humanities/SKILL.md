## 적용 대상

- 영문 학술지(Journal of Moral Education, Ethics, Philosophical Studies 등) 투고 원고
- 영문 학술 단행본의 chapter
- 국제학회 발표문(conference paper) 및 proceedings
- 영문 dissertation·thesis 챕터

> 본 스킬은 **인문학(철학·윤리학·교육철학·문학·역사·종교학)** 을 1차 대상으로 한다.
> 사회과학(IMRaD 구조 중심) 또는 자연과학과는 hedging 빈도, 1인칭 사용, 시제 관습이 다르다.

## 1. 학술 영어 5대 원칙

| 원칙 | 의미 | 위반 예 → 권장 예 |
|------|------|------------------|
| **Clarity (명확성)** | 한 문장 = 한 명제 | "There are many things that we could say about X" → "X has three main features" |
| **Concision (간결성)** | 군더더기 제거 | "Due to the fact that" → "Because" / "In order to" → "To" |
| **Precision (정밀성)** | 모호한 일반화 회피 | "Many scholars think..." → "Three recent studies (Smith, 2020; Jones, 2021; Lee, 2022) argue..." |
| **Objectivity (객관성)** | 감정·과장 자제 | "It is obvious that..." → "The evidence suggests that..." |
| **Formality (형식성)** | 구어체·축약형 회피 | "doesn't / can't / won't" → "does not / cannot / will not" |

> 인문학에서는 자연과학 대비 1인칭(I, we) 사용이 허용된다. "I argue", "I propose"는 자연스러운 학술 영어다.

## 2. 시제 사용 규칙 (인문학 표준)

인문학 학술 글쓰기는 **literary present tense** 관습을 따른다 (Vanderbilt Writing Studio).

### 2-1. 선행연구 인용

| 상황 | 시제 | 예시 |
|------|------|------|
| 일반적 연구 동향 (구체적 시점 X) | **현재완료** (present perfect) | "Recent studies **have shown** that virtue ethics is gaining renewed attention." |
| 특정 저자·연도 명시 (구체적 행위) | **단순과거** (simple past) | "Mele (1987) **proposed** a causalist account of intentional action." |
| 저자의 입장이 현재도 유효 | **현재시제** (present simple) | "Davidson **argues** that reasons are causes." |
| 인용한 명제·정의 | **현재시제** | "Smith (2020) **defines** moral integrity as ..." |

> **혼용 가능**: 같은 단락에서 "Mele (1987) **proposed** ... His view **maintains** that..." 처럼 행위는 과거, 명제는 현재로 쓸 수 있다.

### 2-2. 본 논문 주장

| 상황 | 시제 | 예시 |
|------|------|------|
| 논문 자체의 주장·기여 | **현재시제** | "This paper **argues** that ..." / "I **propose** ..." |
| 논문의 진행 안내 | **현재시제** 또는 **미래시제** | "Section 3 **presents** ..." / "I **will demonstrate** that ..." |
| 결론에서 수행 결과 회고 | **현재완료** | "This study **has identified** three conditions for ..." |

### 2-3. 1차 텍스트 인용 (literary present)

문학·철학 1차 텍스트는 "영원한 현재"로 다룬다.

| 상황 | 시제 | 예시 |
|------|------|------|
| 1차 텍스트 안의 명제·서술 | **현재시제** | "Aristotle **states** that *eudaimonia* **is** the highest human good." |
| 텍스트의 등장인물·서사 행위 | **현재시제** | "In the *Republic*, Socrates **questions** Thrasymachus's definition of justice." |
| 저자의 역사적 행위 (집필·출판) | **단순과거** | "Aristotle **wrote** the *Nicomachean Ethics* in the fourth century BCE." |

> **주의**: 같은 문장에서 시제 전환이 3번 이상 되면 두 문장으로 나눈다 (Vanderbilt 권장).

### 2-4. 분석 결과 진술

- 논리적·해석적 결론: **현재시제** ("These considerations **suggest** that...")
- 사례 연구의 발견: **단순과거** ("The analysis **revealed** three patterns...")

## 3. 인문학 핵심 표현 (Academic Phrasebank 기반)

### 3-1. 선행연구 도입 (Introducing the literature)

```
Recent studies have shown that ...
Previous research has established that ...
It has been argued that ...
Several lines of evidence suggest that ...
Building on the work of [Author], ...
A growing body of literature has examined ...
```

### 3-2. 갭 식별 (Identifying the gap)

```
However, little attention has been paid to ...
Despite extensive research, X remains underexplored.
No previous study has investigated ...
The mechanisms that underpin X are not fully understood.
This question has received surprisingly little scholarly attention.
```

### 3-3. 본 논문 주장 (Stating the contribution)

```
This paper argues that ...
I will demonstrate that ...
I propose ...
The present study seeks to address this gap by ...
The aim of this paper is to ...
```

### 3-4. 반박 검토 (Anticipating objections)

```
One might object that ...
Critics may argue that ...
A possible objection here is that ...
While this view has merit, it overlooks ...
Although [Author] maintains that X, I will show that ...
```

### 3-5. 결론 (Concluding)

```
In conclusion, ...
To summarize, ...
These findings suggest that ...
This study contributes to the literature by ...
This paper has argued that ...
```

## 4. Hedging — 학술적 신중성

Hyland (1994)에 따르면 **인문·사회과학은 자연과학보다 hedging을 더 많이** 사용한다. 단, 핵심 주장은 단호하게 진술해야 약한 논문으로 읽히지 않는다.

### 4-1. 강도 표현 (modal verbs — 강 → 약)

| 강도 | 표현 | 예시 |
|------|------|------|
| 강 | **will / does / is** | "This account **resolves** the puzzle." |
| 중상 | **can / should** | "This **can** be explained by ..." |
| 중 | **may / might** | "This **may** suggest that ..." |
| 약 | **could / would** | "It **could** be argued that ..." |

### 4-2. 빈도 표현 (frequency adverbs)

```
always > often / frequently > sometimes / occasionally > rarely > never
```

### 4-3. 인지 동사·부사 (epistemic verbs/adverbs)

```
seems / appears / suggests / indicates / tends to
arguably / presumably / possibly / likely / perhaps
```

### 4-4. 단정 회피 패턴

| 단정 (피하기) | Hedged (권장) |
|---------------|--------------|
| "X is true." | "X **appears to be** the case." |
| "This proves Y." | "This **suggests** Y." |
| "Everyone agrees that ..." | "There **is broad agreement** that ..." |
| "X always causes Y." | "X **tends to** cause Y." |

### 4-5. 균형 — 과도한 hedging은 약한 주장이 된다

```
[과도] It might possibly be argued that there could perhaps be some 
       evidence that X may sometimes seem to suggest Y.
       
[권장] The evidence suggests that X often correlates with Y.
```

> **원칙**: 핵심 주장(thesis)은 단호하게, 보조 주장은 hedged로.

## 5. Signposting — 논증 흐름 안내

### 5-1. 순서 (Sequence)

```
First, ... Second, ... Third, ... Finally, ...
To begin with, ... / Initially, ...
```

### 5-2. 추가 (Addition)

```
Furthermore, ... / Moreover, ... / In addition, ...
Another significant aspect is ...
```

### 5-3. 대조 (Contrast)

```
However, ... / Nevertheless, ... / Nonetheless, ...
In contrast, ... / On the other hand, ...
Unlike Smith, Jones (2014) has argued ...
```

### 5-4. 인과 (Causation)

```
Therefore, ... / Consequently, ... / Thus, ...
As a result, ... / It follows that ...
```

### 5-5. 예시 (Exemplification)

```
For instance, ... / For example, ...
To illustrate, ... / A case in point is ...
```

### 5-6. 회귀·전환 (Returning / Turning)

```
Returning to the question of X, ...
Turning now to ..., ...
Having defined X, I now move on to discuss ...
```

> 출처: [Manchester Academic Phrasebank — Signalling Transition](https://www.phrasebank.manchester.ac.uk/)

## 6. 문장 구조 (인문학 권장)

### 6-1. 문장 길이

- 평균 **20~25 단어** 권장
- 35 단어 초과 시 분할 검토
- 짧은 문장(10단어 이하)과 긴 문장 혼합으로 리듬 확보

### 6-2. 능동태 vs 수동태

| 상황 | 권장 태 | 예시 |
|------|---------|------|
| 행위자 명시·강조 | **능동** | "Aristotle **argues** that ..." |
| 행위자 불명·일반론 | **수동** | "It **has been observed** that ..." |
| 본 논문 주장 | **능동 + 1인칭** | "**I argue** that ..." |
| 객관성 강조 (실험·관찰) | **수동** | "The data **were analyzed** using ..." |

> **주의**: 인문학에서 모든 문장을 수동태로 쓰면 무거워진다. 능동태를 기본으로, 객관성이 필요한 곳에만 수동태를 쓴다.

### 6-3. 명사화 (Nominalization)

학술 영어는 동사보다 명사 형태를 자주 쓴다. 단, 과도하면 가독성이 떨어진다.

```
[적정]    "The analysis of these texts reveals three patterns."
[과도]    "The undertaking of an analysis of the examination of these
          texts is the revelation of three patterns."
```

### 6-4. 주어-동사 거리

주어와 본동사 사이가 멀면 가독성이 떨어진다.

```
[나쁨] "The argument that Aristotle, in contrast to Plato who emphasized 
       the Forms, develops in the Nicomachean Ethics regarding eudaimonia 
       is compelling."

[좋음] "Aristotle's argument about eudaimonia in the Nicomachean Ethics 
       is compelling. Unlike Plato, who emphasized the Forms, Aristotle 
       grounds the good life in human function."
```

## 7. 한국어 화자 특화 — 흔한 실수

한국어는 관사가 없고 통사 구조가 다르므로 다음 실수가 빈발한다 (영어교육학 연구).

### 7-1. 관사 (Articles) — 한국어 화자 1순위 실수

| 실수 | 수정 | 설명 |
|------|------|------|
| "I read book yesterday." | "I read **the** book yesterday." | 특정 책: the |
| "He is professor." | "He is **a** professor." | 직업: a/an |
| "The honesty is a virtue." | "Honesty is a virtue." | 추상명사 일반론: 무관사 |
| "Aristotle wrote a Nicomachean Ethics." | "Aristotle wrote **the** *Nicomachean Ethics*." | 특정 작품: the |

**규칙 요약**:
- 처음 언급 가산명사 단수 → **a / an**
- 화자·청자가 아는 특정 대상 → **the**
- 추상명사 일반론·복수 일반론 → **무관사**
- 작품·서적 제목 (the로 시작 안 하면) → 인용 시 the 붙임 (the *Republic*, the *Nicomachean Ethics*)

### 7-2. 단·복수 일치

| 표현 | 단수/복수 | 예시 |
|------|----------|------|
| **data** | 학술 영어에서 **복수** 권장 (APA, Cell Press 표준) | "These data **suggest** ..." |
| **research** | 불가산 | "Research **shows** that ..." (NOT "researches show") |
| **literature** (선행연구) | 불가산 | "The literature **indicates** ..." |
| **evidence** | 불가산 | "Evidence **supports** ..." |
| **criteria / phenomena** | 복수 (단수: criterion / phenomenon) | "These criteria **are** ..." |

> **주의 (DISPUTED 정정)**: 한국어 화자가 흔히 "data is"로 쓰는 경우가 있으나, 학술 출판(APA, Taylor & Francis, 인문·사회과학 표준)에서는 **data를 복수로 처리**하는 것이 안전하다. 일부 IT·구어체에서는 단수 사용이 늘었지만, 인문 학술지 투고에서는 plural을 권장한다. ([APA / Cell Press 가이드](https://crosstalk.cell.com/blog/is-data-plural))

### 7-3. 번역투·중복 표현

| 한국어식 영어 | 학술 영어 |
|--------------|----------|
| "However, but ..." (중복) | "However, ..." 또는 "But ..." 중 하나 |
| "Until now, ..." | "**To date**, ..." / "So far, ..." |
| "It is thought that ~" 과용 | 능동태로 — "Scholars argue that ..." |
| "There is no doubt that ..." | "**Arguably**, ..." (hedged) |
| "As we all know, ..." | 삭제 또는 "As [Author] has noted, ..." |
| "I want to say that ..." | "**I argue that** ..." |
| "very important" | "**crucial / critical / significant**" |
| "a lot of studies" | "**numerous studies / a substantial body of research**" |

### 7-4. 전치사

| 실수 | 수정 |
|------|------|
| "research **about** X" | "research **on** X" |
| "discuss **about** X" | "discuss X" (about 불필요) |
| "different **with**" | "different **from**" |
| "in the case **of** X" | OK / "in X's case" 도 가능 |

## 8. 고전·1차 텍스트 인용 영어 표현

### 8-1. 인용 도입

```
As Aristotle argues in the Nicomachean Ethics (NE VII.3, 1147a24-b5), ...
According to Davidson (1980, p. 41), reasons are causes.
Davidson (1980, p. 41) maintains that ...
In the Republic (Book IV, 435c), Plato divides the soul into three parts.
```

### 8-2. 표준 약어·인용 형식

| 텍스트 | 약어 | 예시 인용 |
|--------|------|-----------|
| Aristotle, *Nicomachean Ethics* | NE 또는 EN | (NE 1103a17) |
| Plato, *Republic* | Rep. | (Rep. 514a) |
| Kant, *Critique of Pure Reason* | KrV | (KrV A51/B75) |
| Aristotle, *Politics* | Pol. | (Pol. 1253a) |

### 8-3. 외국어 (그리스어·라틴어·독일어) 표기

- **이탤릭** + **음역(transliteration)** + **(영어 번역)** 병기
- 그리스어: *akrasia* (weakness of will), *eudaimonia* (flourishing/happiness)
- 라틴어: *summum bonum* (the highest good), *a priori*
- 독일어: *Geist* (spirit/mind), *Dasein* (being-there)

```
Aristotle's notion of *akrasia* — typically translated as "weakness of will" — 
denotes acting against one's better judgment.
```

### 8-4. 직접 인용 부호

- **40 단어 미만**: 본문 안에 큰따옴표 ("...")
- **40 단어 이상**: block quotation (들여쓰기, 따옴표 없음, APA 7판 기준)
- **인용 안의 인용**: 작은따옴표 ('...')
- **생략**: [...] (대괄호 안 점 세 개)
- **삽입·설명**: [insertion] (대괄호)

## 9. Discussion · Conclusion 핵심 표현

### 9-1. 발견 요약

```
This paper has argued that ...
The findings of this study suggest that ...
Three main conclusions can be drawn from this analysis: ...
```

### 9-2. 학문적 기여

```
This study contributes to the literature by ...
The present analysis extends previous work on X by ...
This paper offers a novel account of ...
```

### 9-3. 한계 명시

```
A limitation of this study is that ...
The scope of this paper is restricted to ...
Further empirical work would be needed to confirm ...
```

### 9-4. 향후 연구

```
Future research should investigate ...
A promising avenue for further inquiry is ...
This raises questions that warrant further exploration.
```

## 11. 영문 학술지 투고 전 체크리스트

```
[ ] 시제 일관성 확인 (선행연구·1차 텍스트·본 논문 주장 시제 구분)
[ ] 핵심 주장은 단호한 modal, 보조 주장은 hedged
[ ] Signposting으로 단락 간 흐름 명시
[ ] 평균 문장 길이 20~25 단어 (35 초과 분할)
[ ] 관사·단복수 일치 점검 (특히 the / a / an)
[ ] 번역투 표현 제거 (However but, Until now, very important 등)
[ ] 1차 텍스트 인용 형식 (저자·작품·표준 약어)
[ ] 외국어 단어는 italics + 음역 + 영어 번역 병기
[ ] 학술지 referencing style 준수 (JME = APA 7판)
[ ] AI 사용 시 disclosure 정책 확인
```
