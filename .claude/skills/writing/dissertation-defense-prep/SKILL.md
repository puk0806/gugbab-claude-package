## 0. 사용 시점

- 본심사 일정이 1~3개월 앞으로 확정된 시점부터
- 예비심사(초심) 통과 직후 → 본심사(종심) 준비 단계
- 디펜스 1주일 전 / 1일 전 / 당일 체크리스트 점검 시
- 본심사 통과 후 수정보완 응답서 작성 시

## 1. 한국 vs 국제 디펜스 형식 비교

### 한국 박사 본심사 (일반적인 패턴)

| 항목 | 일반 범위 | 비고 |
|------|----------|------|
| 발표 시간 | 30~40분 | 학과별 편차, 일부 학과 20~30분 |
| 질의응답 | 60~90분 | 공개 Q&A 후 비공개 토론 추가 |
| 심사위원 | 5인 (위원장 1, 부위원장 1, 위원 3) | 서울대 기준, KAIST는 4인 |
| 합격 기준 | 위원의 4/5 이상 찬성 | 서울대 종심·구술고사 70점 이상 |
| 진행 방식 | 공개 발표 → 질의응답 → 심사위원 비공개 평가 | 학생·지도교수 퇴실 후 심사요지 작성 |

### 한국 석사 학위논문 심사

| 항목 | 일반 범위 |
|------|----------|
| 발표 시간 | 15~20분 |
| 질의응답 | 20~40분 |
| 심사위원 | 3인 |

### 미국 PhD Defense (Penn State 기준)

- **2단계 구성**: Public oral presentation + Closed committee discussion
- 위원회 구성원 **2/3 이상 찬성**으로 통과
- 최소 3인 이상 물리적 출석 (지도교수 포함), 1인까지 원격 가능
- 시험 일정은 **최소 2주 전 공지** 필수

### 영국·네덜란드 Viva Voce

- **1:1 변론 형식**, 2~4시간 진행 (평균 2시간)
- 심사위원: 내부 1인 + 외부 1인 (2인 이상)
- **비공개**(non-public) 진행, 지도교수는 참여하지 않거나 참관만
- 공개 발표가 없는 경우가 많음 (일부 네덜란드는 공개 의식 별도)

> **주의:** 위 수치는 일반적 패턴이며, 본인 소속 대학원의 공식 시행세칙이 우선한다. 서울대 약학대·전기정보공학부 등은 학기마다 별도 공고를 게시한다.

## 2. 발표 자료(PPT) 표준 구성

### 박사 30~40매 권장 구성

| 섹션 | 슬라이드 수 | 시간 비중 | 주요 내용 |
|------|:----------:|:--------:|----------|
| 표지 | 1 | - | 제목, 이름, 지도교수, 심사일 |
| 목차 | 1 | - | 발표 흐름 안내 |
| 연구 배경·문제 정의 | 3~5 | 도입부 25% | 학문적·사회적 필요성, 문제 진술 |
| 연구 목적·질문 | 1~2 | | RQ/가설 명시 |
| 선행연구 검토 | 3~5 | | 이론적 공백, 기여 위치 |
| 이론·방법론 | 3~5 | 핵심 50% | 분석 틀, 데이터, 측정 방법 |
| 핵심 논증·발견 | 10~15 | | 본 연구의 가장 강력한 부분 |
| 결론·기여·한계 | 3~5 | 결론 25% | 학술적·실무적 기여, 후속 연구 |
| 백업 슬라이드 | 5~10 | (Q&A용) | 세부 표·통계·반론 대응 자료 |

### 슬라이드당 평균 시간

- **1슬라이드 = 약 1분** (도입·결론은 30초~1분, 핵심 발견은 1.5~2분)
- 30분 발표 = 본 슬라이드 25~30매가 적정
- 백업 슬라이드는 발표 시간에 포함하지 않음

### 슬라이드 디자인 원칙

- 본문 폰트 24pt 이상 (Arial, Calibri, 맑은 고딕 등)
- 한 슬라이드 한 메시지 원칙
- 슬라이드를 그대로 읽지 말고 구두 설명으로 확장
- 애니메이션·전환 효과 최소화 (디펜스는 학술 발표이지 마케팅이 아니다)
- 챕터 번호·진행 표시(1/8, 2/8 ...)로 위치 안내

## 3. 발표 시간 관리

### 시간 배분 가이드라인

```
30분 발표 기준
  도입부 (배경·문제·목적·선행연구):   7~8분  (25%)
  본론 (방법론·핵심 발견):           15분   (50%)
  결론 (기여·한계·후속연구):          7~8분  (25%)
```

### 리허설 규칙

- 최소 **3회 이상 풀 리허설** (시간 측정 필수)
- 1회는 지도교수 또는 동료 박사과정생 앞에서
- 1회는 비전공자 앞에서 (전문 용어 노출도 점검)
- 1회는 본인 단독 + 녹화·재청취

### 시간 초과 방지

- 슬라이드별 목표 시간을 노트에 기록
- 도입부에서 평균 30초 늦어지면 본론 슬라이드 1매 압축 결정
- 발표 시작 후 7분 시점에 도입부 종료가 정상 페이스

## 4. 질의응답 준비

### 예상 질문 50개 사전 준비

연계 에이전트: `research/defense-question-simulator` (있을 경우)

#### 카테고리별 분류 (각 8~10개)

| 카테고리 | 질문 예시 |
|---------|----------|
| **연구 주제** | "왜 이 주제를 선택했는가?", "유사 주제 중 이것의 차별성은?" |
| **방법론** | "왜 X 방법이 아니라 Y 방법을 선택했는가?", "표본 크기 선정 근거는?" |
| **해석·결과** | "이 결과의 통계적 유의성은?", "예상과 다른 결과가 나온 이유는?" |
| **논증 구조** | "이 주장의 가장 약한 고리는 무엇인가?", "반대 견해는?" |
| **적용·일반화** | "다른 맥락에 일반화 가능한가?", "실무 적용 가능성은?" |
| **한계·후속** | "이 연구의 가장 큰 한계는?", "다음에 무엇을 연구할 것인가?" |

#### 질문 발굴 방법

- 지도교수와 1:1 미팅에서 받은 코멘트 모두 기록
- 예비심사(초심) 위원 코멘트 정리
- 본인 논문의 모든 "그러나", "그럼에도", "한편" 표현 점검
- 본인 분야 최근 5년 박사학위 디펜스 후기 (김박사넷 등 커뮤니티)

### "잘 모르겠습니다" 표현법

> 모르는 것을 추측으로 답하는 것이 가장 큰 감점 요인이다.

#### 권장 표현

- "좋은 질문 감사합니다. 그 부분은 본 연구에서 깊이 다루지 못했습니다. 추후 연구에서 검토하겠습니다."
- "현재 자료로는 단정하기 어렵습니다. 다만 X 관점에서는 ... 이렇게 해석할 여지가 있습니다."
- "지적해주신 부분은 미처 살펴보지 못했습니다. 본심사 후 수정 과정에서 반영하겠습니다."

#### 금지 표현

- "그건 제 연구 범위가 아닙니다" (방어적·회피)
- 추측성 답변 ("아마 ... 일 것 같습니다" 단독 사용)
- 침묵 후 무관한 답변

## 5. 답변 기법

### PREP 구조 (Point–Reason–Example–Point)

```
Point   : 결론을 먼저 한 문장으로
Reason  : 그 근거를 1~2문장으로
Example : 본 연구의 데이터·인용·사례
Point   : 결론을 다시 짧게 재확인
```

**예시 답변:**
> "(P) 본 연구는 X 방법이 적합하다고 판단했습니다.
> (R) 왜냐하면 표본의 규모와 종속변수 분포가 정규성을 만족하지 않았기 때문입니다.
> (E) 실제로 4장 표 3에서 Shapiro-Wilk 검정 p=0.02로 정규성이 기각되었고, 이에 비모수 검정을 채택했습니다.
> (P) 따라서 X 방법 선택은 데이터 특성상 타당하다고 봅니다."

### 답변 시간

- **30~60초가 표준** (1분 초과 금지)
- 길어지면 위원이 끊고 다른 질문으로 넘어감
- 핵심을 먼저, 부연 설명은 위원이 추가 질문할 때

### 반박 인정 + 재질문 활용

- 위원 지적이 타당하면 즉시 인정: "예, 그 부분은 한계로 인지하고 있습니다."
- 의도가 불명확한 질문은 재확인: "질문하신 의도가 X에 대한 것인지, Y에 대한 것인지 확인 부탁드려도 될까요?"
- 강한 반박을 받았을 때: 동의할 수 있는 부분 먼저 인정 → 본인 입장의 근거 제시 → 추가 검토 약속

### 절대 금지

- 위원 의견을 강하게 부정 ("그건 틀렸습니다")
- 감정적 반응 (한숨, 표정 변화, 목소리 높임)
- 다른 위원·지도교수에게 책임 전가

## 6. 실무 행정

### 사전 준비물 체크리스트

| 항목 | 비고 |
|------|------|
| 인쇄본 논문 | 심사위원 수 + 본인용 1부 (학과별 사전 제출본과 별개) |
| 발표 자료 USB 백업 | 2개 이상 (PDF + PPTX 동시 저장) |
| 노트북·전원 어댑터 | HDMI/USB-C 등 필요 어댑터 모두 |
| 클라우드 백업 | Google Drive, OneDrive 등 사전 업로드 |
| 발표 노트(스크립트) | 인쇄본 또는 듀얼 모니터 |
| 포인터·리모컨 | 배터리 여분 |
| 필기구·메모지 | Q&A 메모용 |
| 물·간단한 간식 | 90분 이상 진행 대비 |
| 학위논문 인쇄본 | 위원 요청 시 즉시 펼칠 수 있도록 |

### 복장

- 정장 권장 (남: 셔츠+넥타이+재킷, 여: 정장 또는 단정한 비즈니스 캐주얼)
- 학과 관행 우선 (이공계는 비즈니스 캐주얼 허용 사례 있음)
- 본심사 1~2주 전 미리 준비, 당일 옷 점검

### 시간

- **30분 전 도착**: 장비 셋업, 마이크·프로젝터 점검, 화장실
- 입장 직전 5분: 심호흡, 첫 30초 도입부 멘탈 리허설
- 발표 종료 후 비공개 평가 시간(15~30분) 동안 대기실에서 대기

### 녹음

- 본인 학습용 녹음은 **반드시 심사위원 전원 동의 후** 실시
- 동의 없는 녹음은 학사·법적 분쟁 소지
- 동의 받았다면 스마트폰 앱(Google 녹음, iPhone 음성메모) 사용

## 7. 수정·보완 단계 (본심사 후)

### 일반적인 일정

- 본심사 후 수정 의견 정리: **1~4주** (대학·학과별 편차)
- 수정본 + 응답서 위원 회람 → 위원장 최종 인준
- 인준 후 도서관 dCollection 등록 → 인쇄본 제출

> **주의:** 일부 대학은 보존용 논문 제출 후 명백한 오탈자 외 정정 불가. 제출 전 충분히 검토 필수.

### Point-by-Point Response 구조

`writing/journal-submission-response` 스킬과 동일한 형식 (해당 스킬이 있을 경우 참조):

```
[심사위원 1 코멘트 1]
지적 사항: (위원의 지적을 그대로 인용)
답변: (수정 내용 또는 미수정 사유)
수정 위치: 본문 p.42, 표 3-2
```

### 답변 작성 원칙

- 위원의 모든 코멘트에 대해 빠짐없이 응답
- "감사합니다"로 시작, 수정 내용을 구체적으로 명시
- 미수정 항목은 학술적 근거를 들어 정중히 설명
- 페이지·표·그림 번호로 수정 위치 명시

## 8. 심사 통과 후 행정

### 학위논문 등록

| 항목 | 내용 |
|------|------|
| dCollection 등록 | 소속 대학 도서관 학위논문 제출 시스템 (예: dcollection.yonsei.ac.kr) |
| 제출 파일 | PDF 본문 + 인준지 PDF (서명·날인 포함) 별도 |
| 제출 형식 | 본문 PDF 내 인준지는 위원 이름만, 서명·날인은 별도 PDF |
| 인쇄 보존본 | 도서관·학과·본인용 (학과별 부수 상이) |

### 학위 수여 신청

- 소속 대학원 학사 시스템에서 학위 수여 신청
- 졸업사정 회의 일정 확인 (학기말 마감 엄수)
- 외국 대학 박사학위인 경우: **한국연구재단 외국박사학위 신고** 필수 (귀국·취득 후 6개월 이내, https://dr.nrf.re.kr)

> **주의:** 국내 대학 박사학위는 한국연구재단 별도 신고 의무 없음. 외국 박사학위만 해당.

## 9. 심리 준비

### 마음가짐

- 디펜스는 **시험이 아니라 학술 대화**다. 본인이 그 주제의 가장 깊은 전문가.
- 심사위원은 적이 아니라 **동료 학자**. 비판도 학문적 발전 의도다.
- 통과율은 일반적으로 매우 높다(대부분 수정 조건부 통과). 떨어지는 케이스는 극히 드물다.
- 본인이 작성한 논문이고 본인이 가장 많이 읽었다. 자신감의 근거.

### 디펜스 1주일 전 멘탈 관리

- 새로운 자료 추가 학습 금지 (혼란만 가중)
- 충분한 수면 (심리학 연구상 단기 휴식이 기억·문제해결력 향상)
- 가벼운 운동, 명상, 산책으로 긴장 완화
- 디펜스 성공 장면 시각화 (1~2분, 매일)

### 당일 심리 안정 기법

- 도착 후 4-7-8 호흡법 (4초 들이쉬고 7초 멈추고 8초 내쉬기) 3회
- 첫 슬라이드 인사말은 **반드시 외워서** 진행 (가장 떨리는 30초)
- 위원의 어려운 질문 앞에서 5초 침묵은 정상 (생각 정리 시간)

| **모르는 답변에 추측** | 솔직함 부족, 압박감 | "추후 보완하겠습니다" 표현 연습 |
| **방어적·감정적 반응** | 비판을 인신공격으로 오해 | 위원=동료 학자라는 인식 전환 |
| **인쇄본 부족 / USB 분실** | 사전 점검 부재 | 인쇄본 +1부, USB 2개, 클라우드 동시 백업 |
| **발표 자료 노트북 호환 문제** | 다른 환경에서 미점검 | 심사실 프로젝터 사전 테스트 |
| **슬라이드를 그대로 읽음** | 발표 연습 부족 | 슬라이드는 키워드, 입으로는 확장 설명 |
| **선행연구만 길게 설명** | 본인 기여 강조 부족 | 본 연구의 차별성을 도입부에서 명시 |
| **위원 코멘트 메모 안 함** | Q&A 집중력 부족 | 동행자 또는 본인이 직접 메모 |
