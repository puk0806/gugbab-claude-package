## 1. 큰 주제를 RQ로 좁히는 5단계 (Booth/Colomb/Williams 5판 기반)

> Booth 5판은 4부 흐름을 사용한다: **Interest → Topic → Question → Problem**.
> 본 스킬은 한국 대학원 실무에 맞춰 "최종 RQ 확정"을 별도 단계로 추가한 **5단계**로 재구성한다.

### 단계 1: 관심사(Interest) 흩뿌리기

분야 경계 없이 "끌리는 모든 것"을 30개 이상 적어본다. 검열 금지.

```
나쁜 예: "akrasia"  ← 이미 좁혀진 주제. 1단계가 아님
좋은 예:
- 학생들이 시험 직전 핸드폰 못 놓는 현상
- 다이어트 결심하고 야식 먹는 나
- "아는데 못 한다"는 말의 철학적 분석
- 도덕 수업이 행동 변화에 실패하는 이유
- 아리스토텔레스의 "자제력 없음" 개념
- ...
```

### 단계 2: 광범위한 주제(Topic)로 묶기 — Booth Ch.3

여러 관심사가 가리키는 **하나의 묶음 영역**을 찾는다. 4가지 narrowing 관점(Booth) 중 1개 이상을 적용한다.

| 관점 | 질문 |
|------|------|
| 부분-전체 | "이 주제의 어떤 측면에 끌리는가?" |
| 역사적 변화 | "이 주제가 시간에 따라 어떻게 변했는가?" |
| 범주·특성 | "이 주제는 어떤 종류로 나뉘는가?" |
| 가치 평가 | "이 주제의 무엇이 좋은가/문제인가?" |

```
관심사 묶음: "아는 것과 행하는 것의 불일치"
→ Topic (부분-전체 관점): "도덕적 앎과 도덕적 행위의 간극에 관한 아리스토텔레스적 분석"
```

### 단계 3: 질문(Question)으로 변환

Topic은 명사, **Question은 의문문**이다. **descriptive 질문(무엇/누구)에서 멈추지 말고 interpretive/analytical 질문(왜/어떻게/무슨 의미)으로 진행**한다 (Booth 5판 Ch.3).

```
descriptive (약함):
- "아리스토텔레스는 akrasia를 어떻게 정의했는가?"

interpretive/analytical (강함):
- "NE 7권의 propeteia/astheneia 구분이 도덕교육의 어떤 단계 구분과 대응되는가?"
- "왜 현대 도덕교육 연구는 akrasia 유형 구분을 활용하지 못하는가?"
```

### 단계 4: 문제(Problem)로 격상 — Booth Ch.4

좋은 RQ는 **답을 찾는 것이 누군가의 이해를 바꾸는 문제**여야 한다. Booth의 3단계 템플릿:

```
Topic:        I am studying ______________
Question:     because I want to find out what/why/how ______________
Significance: in order to help my reader understand ______________
```

이 템플릿을 통과하지 못하면 RQ가 아니라 사실 조사 과제다. 이때 적용하는 것이 **"so what" 테스트**: 읽는 사람이 "그래서 어떻게 할 건데?"를 "뭔가 문제네"로 전환할 때까지 질문을 반복한다.

| 문제 유형 | 정의 | 인문학 적용 |
|-----------|------|-------------|
| Practical Problem | 세상의 비용·고통을 야기하는 조건 | 도덕수업 후 학생 행동이 변하지 않는 현상 |
| Conceptual/Research Problem | "우리가 X를 충분히 이해하지 못한다"는 결함 | NE 7권 akrasia 분류 체계가 한국어 학계에서 일관되게 해석되지 않음 |

> 인문학 학위논문 RQ는 **Conceptual Problem이 1차이고, Practical Problem은 "기여" 섹션에서 부차적으로 연결**한다.

### 단계 5: 최종 RQ 확정 — 1개 main RQ + 2~3개 sub-RQ

석사 논문: main RQ 1개 + sub-RQ 2~3개
박사 논문: main RQ 1개 + sub-RQ 3~5개

```
Main RQ:
  NE 7권 1145b21–1147b19에서 아리스토텔레스가 제시한 akrasia의
  propeteia/astheneia 구분은, 현대 한국 중등 도덕교육의 어떤 교수학습
  국면에 차별적으로 적용될 수 있는가?

Sub-RQ 1: NE 7권의 두 유형은 텍스트 내적으로 어떻게 정합적으로 구분되는가?
Sub-RQ 2: 두 유형 구분을 받아들이는/거부하는 주요 주석가들의 논거는 무엇인가?
Sub-RQ 3: 2015·2022 도덕과 교육과정의 어느 성취기준이 두 유형 중
          어느 쪽 처방을 요구하는가?
```

## 2. 인문학 연구계획서 표준 8개 섹션

> 한국 대학원 석/박사 학위논문 연구계획서 기준 표준 골격.
> 학과·지도교수에 따라 6~10개로 변형되지만 핵심 8개는 거의 일치.

| # | 섹션 | 핵심 질문 | 분량(석사 기준) |
|---|------|-----------|-----------------|
| 1 | 연구 배경·필요성 | 왜 지금 이 연구가 필요한가 | 2~3쪽 |
| 2 | 선행연구 검토 | 무엇이 이미 밝혀졌고 무엇이 빈 칸인가 | 3~5쪽 |
| 3 | 연구문제·가설 | 정확히 무엇을 묻는가 | 1~2쪽 |
| 4 | 이론적 틀 | 어떤 렌즈로 분석하는가 | 2~3쪽 |
| 5 | 연구방법·범위 | 어떤 자료를 어떻게 다루는가 | 2~3쪽 |
| 6 | 연구 일정 | 학기별 무엇을 끝내는가 | 0.5~1쪽 |
| 7 | 예상 기여 | 답을 찾으면 누구의 무엇이 바뀌는가 | 1~2쪽 |
| 8 | 참고문헌 | 어디에서 출발하는가 | 2~3쪽 |

### 섹션 1: 연구 배경·필요성

```
구조:
  (1) 광의의 학문적·실천적 맥락 (2~3문단)
  (2) 협의의 문제 상황 — Booth Practical+Conceptual Problem 명시
  (3) 본 연구가 메우려는 빈 칸 (1문단)
```

> 주의: 시간순으로 학설사를 나열하는 "교과서식 도입"은 금지. 반드시 **빈 칸 진단**으로 끝내야 한다.

### 섹션 2: 선행연구 검토 (Literature Review)

선행연구를 **요약하지 말고 평가**하라 (Booth 5판 Ch.5–6).

```
구조 옵션:
  A. 주제별 (Thematic) — 인문학 표준
     예: "akrasia 정의 논쟁" / "유형 구분 논쟁" / "도덕교육 적용 시도"
  B. 시기별 (Chronological) — 사상사 연구일 때만
  C. 학파별 (School) — 비교철학 연구일 때
```

선행연구 검토 마지막에 반드시 **"본 연구의 차별성" 단락**을 둔다. (한국 학위논문 심사 관행)

### 섹션 3: 연구문제·가설

위 5단계로 도출한 main RQ + sub-RQ를 명시. 가설은 인문학에서 "예상 답변" 또는 "잠정 테제(working thesis)"로 표현.

```
잠정 테제 예시:
  "NE 7권의 두 유형 구분은 도덕적 동기-숙고 단계의 차별적 처방을
   요구하며, 이는 현행 도덕과 교육과정이 단일 처방 모델에 갇혀
   있다는 비판을 가능케 한다."
```

### 섹션 4: 이론적 틀 (Theoretical Framework)

선행연구 검토와 분리되는 별도 섹션. 분석에 동원할 **렌즈**를 명시.

```
인문학 이론적 틀의 3가지 형태:
  (a) 1차 텍스트 자체를 틀로 사용 — "NE 7권의 분석 도구를 그대로 적용"
  (b) 2차 주석가의 해석 틀 채택 — "Burnyeat 1980 / Broadie 1991의 해석에 따른다"
  (c) 외부 이론을 도입 — "Kohlberg 도덕발달 단계론을 비교 틀로 사용"
```

### 섹션 5: 연구방법·범위

인문학 연구의 5가지 방법 유형 (이 스킬 §3 참조). 범위는 반드시 **텍스트 범위 + 시기 범위 + 언어판본 범위**를 명시.

```
범위 예시:
  - 1차 텍스트: NE 7권 (1145a15–1154b34), Bywater 1894 OCT판 기준
  - 한국어 번역 대조: 강상진·김재홍·이창우 역 (2011), 이창우 역 (2006)
  - 2차 문헌 시기: 1963~2025 (Davidson "How Is Weakness of the Will Possible?" 이후)
  - 도덕교육 적용 범위: 2015·2022 개정 도덕과 교육과정
```

### 섹션 6: 연구 일정

석사 4학기 / 박사 6~8학기 기준. 각 학기 끝에 **검증 가능한 산출물**을 명시.

```
석사 4학기 예시:
  1학기: 1차 텍스트 정독 + 선행연구 80편 정리 → 선행연구 검토 초고
  2학기: 이론적 틀 확정 + 분석 챕터 1 → 발표 1회
  3학기: 분석 챕터 2~3 → 중간 발표
  4학기: 도덕교육 적용 챕터 + 결론 → 심사
```

### 섹션 7: 예상 기여

3가지 기여를 분리해서 작성:

| 기여 유형 | 청자 | 예시 |
|-----------|------|------|
| 학문적 기여 | 학계 동료 연구자 | "NE 7권의 두 유형 구분에 대한 한국어권 첫 정밀 분석 제공" |
| 실천적 기여 | 현장 교사·교육과정 개발자 | "도덕수업 단계별 처방 설계의 이론적 근거 제공" |
| 방법론적 기여 | 후속 연구자 | "고전 텍스트–현행 교육과정 대응 분석 모형 제시" |

### 섹션 8: 참고문헌

- 1차 텍스트(원어판 + 번역판) → 2차 문헌 → 교육과정 자료 순으로 분류
- 인문학은 **각주 인용 시스템(Chicago Notes-Bibliography)** 또는 본문 인용(Author-Date) 중 학과 관행 따름
- 도덕윤리교육 분야는 일반적으로 **각주 시스템** 우세

## 3. 인문학 RQ 5유형 (도덕윤리교육 분야)

| 유형 | 핵심 동사 | 산출물 | akrasia 적용 예시 |
|------|-----------|--------|-------------------|
| **개념분석형** | 구분한다·정의한다 | 개념의 정합적 해명 | "NE 7권의 akrasia와 *enkrateia* 부재 상태를 어떻게 정합적으로 구분할 것인가" |
| **사상사형** | 추적한다·재구성한다 | 사상의 변천 지도 | "Davidson 1963 이후 영미권 akrasia 논쟁의 쟁점은 어떻게 분기되었는가" |
| **비교철학형** | 대조한다·통약한다 | 비교표·차이점 분석 | "아리스토텔레스의 akrasia와 王陽明의 知行合一論은 어떤 점에서 비교 가능한가" |
| **적용/실천형** | 적용한다·재구성한다 | 실천적 처방·교안 | "akrasia 두 유형 구분은 도덕수업의 어떤 단계에 차별적으로 적용되는가" |
| **비판적 검토형** | 비판한다·반박한다 | 기존 해석에 대한 반론 | "현행 도덕교육이 채택한 단일 의지박약 모델은 어떤 점에서 NE 텍스트와 어긋나는가" |

> 한 학위논문은 **주된 유형 1개 + 보조 유형 1~2개** 조합이 일반적이다.
> 예: 적용형(主) + 개념분석형(從)

## 4. 좋은 RQ 체크리스트 — Booth 6항목 + 한국 학위논문 2항목

### Booth/Colomb/Williams 5판 기준 (Ch.3–4)

- [ ] **substantive** — 단순 사실 확인이 아닌, 답이 해석을 요구하는가
- [ ] **contestable** — 합리적 학자가 다른 답을 낼 수 있는가
- [ ] **specific** — 텍스트·시기·범위가 명시되는가
- [ ] **interpretive/analytical** — descriptive에 그치지 않는가 (왜·어떻게·의미)
- [ ] **answerable in scope** — 학위 기간 내 자료로 답할 수 있는가
- [ ] **passes "so what" test** — 답을 찾으면 누군가의 이해가 바뀌는가

### 한국 대학원 학위논문 추가 2항목

- [ ] **선행연구와의 차별성이 한 문장으로 진술 가능한가** (심사 관행상 필수)
- [ ] **지도교수의 전공 영역과 정합적인가** (지도 가능성)

## 5. 지도교수 면담용 1쪽 요약 vs 학과 제출용 본 계획서

### 5-1. 지도교수 면담용 1쪽 요약 (One-pager)

지도교수 1차 면담 시점에 가져가는 **A4 1쪽**.

```
[제목] (잠정)

(1) 한 문장 RQ
    "본 연구는 ___을 묻는다."

(2) Booth 3단계 템플릿 그대로 3줄
    I am studying ___
    because I want to find out ___
    in order to help my reader understand ___

(3) 핵심 1차 텍스트 (3개 이내)
    - ...

(4) 핵심 2차 문헌 5편 (저자-연도만)
    - ...

(5) 예상 분석 챕터 3개 (한 줄씩)

(6) 학기 일정 4줄

(7) 면담에서 묻고 싶은 것 3개 (★)
```

> ★ "묻고 싶은 것"이 핵심이다. 지도교수에게 "검토해주세요"는 무례하다. **구체적 결정 지점**을 가지고 가야 한다.

### 5-2. 학과 제출용 본 계획서 (15~30쪽)

§2의 8개 섹션 전체를 채운 정식 문서. 학과 양식이 있으면 양식 우선.

| 항목 | One-pager | 본 계획서 |
|------|:---:|:---:|
| 분량 | 1쪽 | 15~30쪽 |
| 시점 | 1~2학기 면담 | 종합시험 직후·논문 자격시험 |
| 청자 | 지도교수 1인 | 학과 심사 위원회 |
| RQ 형식 | 1문장 + Booth 3단계 | 8개 섹션 전체 |
| 참고문헌 | 5편 | 50~150편 |
| 수정 빈도 | 면담마다 갱신 | 1~2회 |

## 6. 한국 대학원 vs 미국 dissertation proposal 구조 차이

| 항목 | 한국 대학원 (석/박) | 미국 dissertation proposal |
|------|---------------------|----------------------------|
| 시점 | 종합시험 후 학과 심사 | Comprehensive exam 통과 후 prospectus defense |
| 분량 | 15~30쪽 (인문학 기준) | 30~60쪽 + 1차 챕터 초안 동봉 빈번 |
| 심사 형식 | 학과 위원회 서면 + 발표 1회 | dissertation committee 앞 구두 defense |
| 가설 표현 | "잠정 테제" / "예상 결론" | "working thesis" / "argument I will defend" |
| 선행연구 | 차별성 단락 필수 | "intervention in the field" 단락 |
| 일정 | 학기 단위 명시 | 학기 + month 단위 |
| 윤리 항목 | IRB는 필요 시만 | 인터뷰·설문 시 IRB 승인 사본 동봉 표준 |
| 분량 균형 | 선행연구 비중 高 | 방법론·이론적 틀 비중 高 |

> 도덕윤리교육 분야 한국 학위논문은 미국 모델보다 **선행연구 검토 비중이 더 크고**, "차별성" 단락이 명시적으로 요구된다.

## 7. IRB 심사가 필요한 경우 — 추가 항목

### 7-1. IRB 필요 여부 판정

> 인문학 텍스트 분석만 하는 연구는 IRB 불필요. 그러나 도덕윤리교육 학위논문은 학생·교사 인터뷰·설문이 흔하므로 다수 케이스에서 IRB가 필요하다.

| 연구 형태 | IRB 필요? |
|-----------|:---:|
| 1차·2차 문헌 분석만 | ❌ |
| 교육과정·교과서 분석만 | ❌ |
| 학생/교사 **인터뷰** | ✅ |
| 학생/교사 **설문** | ✅ |
| 학생 작품/일기/응답 텍스트 분석 (식별 가능) | ✅ |
| 수업 관찰·녹화 | ✅ |
| 익명화된 공개 데이터 2차 분석 | 면제 신청 가능 |

### 7-2. 인터뷰·설문 포함 연구계획서 추가 섹션

표준 8개 섹션에 다음 4개를 추가:

| 추가 섹션 | 내용 |
|-----------|------|
| **연구대상자 모집** | 인원, 모집 경로, 포함·배제 기준 |
| **연구대상자 보호** | 동의 절차, 비밀 보장, 식별정보 익명화, 자료 보관·폐기 |
| **위험·이익 평가** | 예상 위험(심리적 부담 포함), 직접·간접 이익 |
| **IRB 심의 계획** | 소속 기관 IRB 신청 시점, 예상 심의 기간(통상 1개월+), 심의 면제·정규 심의 구분 |

### 7-3. 학생을 대상으로 할 때의 추가 고려

> 한국연구재단 「인문사회분야 IRB 가이드라인」: 연구자의 지도를 받는 학생은 **취약한 연구대상자**.

- 자발적 참여 보장 강조 (불참이 성적·평가에 영향 없음 명시)
- 미성년자(고교생 이하)는 **법정대리인 동의서** 추가
- 교사 인터뷰는 학교장 협조 공문 별도

## 8. 흔한 실수 패턴

| 실수 | 왜 문제인가 | 교정 |
|------|-------------|------|
| Topic을 RQ로 착각 ("akrasia 연구") | 의문문이 아님 | 5단계 §3~§5 적용 |
| descriptive 질문에서 멈춤 ("아리스토텔레스는 akrasia를 어떻게 정의했나") | 학위논문 수준 아님 | "왜·어떻게·무슨 의미" 추가 |
| 선행연구 = 시간순 요약 | 평가가 빠짐 | 주제별 그룹핑 + 차별성 단락 |
| "so what" 빠진 RQ | 답이 누구의 이해도 바꾸지 않음 | Booth 3단계 템플릿 통과시키기 |
| 이론적 틀 = 선행연구 재진술 | 분석 렌즈 부재 | §2 섹션 4의 (a)/(b)/(c) 중 1택 |
| 1차 텍스트 판본 미명시 | 인용 정합성 깨짐 | OCT/Loeb/Budé 등 비판본 + 번역판 모두 명시 |
| 인터뷰 있는데 IRB 항목 누락 | 심사 반려 또는 자료 사용 불가 | §7 4개 섹션 추가 |
| 지도교수 면담에 "검토 부탁" | 면담 효율 0 | One-pager + 결정 지점 3개 준비 |
| 연구 일정에 산출물 없음 | 진행 점검 불가 | 학기별 검증 가능한 결과물 명시 |
| 기여 = 막연한 "이해 증진" | "so what" 통과 못 함 | 학문/실천/방법 3가지로 분리 |

## 9. akrasia 학위논문 예시 — 처음부터 끝까지

> 본 스킬을 단일 사례로 끝까지 적용한 모델 케이스. 도덕윤리교육 전공 석사 가정.

### 9-1. 5단계 적용

```
[1단계] 관심사 30개 중 묶이는 영역
  - "아는데 못 한다"는 학생들의 자기보고
  - 도덕수업 효과 측정의 어려움
  - NE 7권의 두 유형 구분이 한국어 학계에서 잘 활용되지 않음
  - 의지박약을 단일 현상으로 다루는 도덕교과서

[2단계] Topic
  "아리스토텔레스 akrasia 분류 체계의 도덕교육적 함의"

[3단계] Question (interpretive)
  "NE 7권의 propeteia/astheneia 구분은 도덕수업 단계 설계에
   어떤 차별적 함의를 가지는가?"

[4단계] Booth 3단계 템플릿
  I am studying NE 7권의 두 akrasia 유형 구분을
  because I want to find out 두 유형이 도덕적 실패의 서로 다른
                              메커니즘을 함축하는지
  in order to help my reader understand 현행 도덕수업이 단일
                              처방 모델에 갇힌 결과 어떤 학생을
                              놓치고 있는가를.

[5단계] 최종 RQ
  Main: NE 1145b21–1147b19의 두 유형 구분은 현행 한국 중등
        도덕과 교육과정의 어떤 교수학습 국면에 차별적으로 적용
        가능한가?
  Sub1: 두 유형의 텍스트 내적 정합적 구분 기준은 무엇인가?
  Sub2: 주요 주석가 4인의 두 유형 해석은 어떻게 분기되는가?
  Sub3: 2022 도덕과 성취기준 중 어느 것이 어느 유형 처방을
        요구하는가?
```

### 9-2. RQ 유형 (§3 분류)

**적용/실천형 (主) + 개념분석형 (從)** — 하이브리드 구조

### 9-3. 좋은 RQ 체크 (§4)

- substantive ✅ (해석 요구)
- contestable ✅ (두 유형 구분 자체가 학계 논쟁)
- specific ✅ (1145b21–1147b19, 2022 교육과정)
- interpretive ✅ (어떻게 적용)
- answerable ✅ (석사 4학기 가능)
- so what ✅ (단일 처방 모델 한계 노출)
- 차별성 ✅ (한국어권 첫 두 유형 구분 기반 적용 분석)
- 지도교수 정합성 ✅ (도덕철학+도덕교육 양립)

### 9-4. 표준 8섹션 핵심 1줄

| 섹션 | akrasia 논문 1줄 요약 |
|------|----------------------|
| 1. 배경·필요성 | 의지박약을 단일 현상으로 처리하는 도덕수업의 한계와 NE 두 유형 구분의 미활용 |
| 2. 선행연구 | 영미권 akrasia 논쟁(주제별 4그룹) + 한국 도덕교육의 적용 시도(차별성: 두 유형 구분 미사용) |
| 3. RQ·테제 | 위 main+3 sub RQ + 잠정 테제(두 유형은 차별적 처방 요구) |
| 4. 이론적 틀 | NE 자체 분석 도구(a) + Burnyeat 1980·Broadie 1991 보조(b) |
| 5. 방법·범위 | 1차 텍스트 정밀독해 + 교육과정 매핑, 1145b21–1147b19, 2015·2022 교육과정 |
| 6. 일정 | 4학기, 학기별 산출물 명시 |
| 7. 기여 | 학문(첫 정밀 분석) + 실천(단계별 처방 근거) + 방법(텍스트–교육과정 매핑 모형) |
| 8. 참고문헌 | OCT판 NE + Bywater + 강상진·김재홍·이창우 + 영미 주석가 + 한국 도덕교육 문헌 |

### 9-5. IRB 필요 여부

본 사례는 **텍스트 분석 + 교육과정 분석만**이므로 IRB 불필요.
다만 만약 "도덕교사 인터뷰를 통한 두 유형 처방 인지도 조사"를 추가하면 §7의 4개 추가 섹션 필요.
