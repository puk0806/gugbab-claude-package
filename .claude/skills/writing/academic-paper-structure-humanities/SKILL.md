# 인문학(철학·교육학) 논문 구조와 인용 표기 가이드

> 소스:
> - [한국철학회 논문투고규정](https://hanchul.org/39)
> - [이화여자대학교 한국문화연구원 작성 지침](https://kcri.ewha.ac.kr/kcri/intro/instructions-for-writing.do)
> - [Chicago Manual of Style 17th Edition (Purdue OWL)](https://owl.purdue.edu/owl/research_and_citation/chicago_manual_17th_edition/cmos_formatting_and_style_guide/chicago_manual_of_style_17th_edition.html)
> - [Chicago Manual of Style 공식 Citation Quick Guide](https://www.chicagomanualofstyle.org/tools_citationguide/citation-guide-1.html)
> - [Bekker numbering (Wikipedia 표준 설명)](https://en.wikipedia.org/wiki/Bekker_numbering)
>
> 검증일: 2026-05-03

> **중요**: 한국 인문학 학술지마다 인용 형식이 크게 다르다. **반드시 투고하려는 학술지의 공식 투고규정을 1순위로 확인**하라. 이 스킬은 가장 널리 통용되는 표준을 정리한 것일 뿐이다.

## 1. 논문 유형별 구조

### 1-1. 학부 졸업논문 / 학기 텀페이퍼

| 항목 | 기준 |
|------|------|
| 분량 | A4 10–20매 (200자 원고지 환산 약 60–120매) |
| 구조 | 단순 3–4단 구조 |
| 초록·키워드 | 보통 생략 (지도교수 요청 시 포함) |
| 참고문헌 수 | 10–20편 정도 |

**권장 단순 구조:**

```
1. 서론 (문제 제기 + 논의 범위 + 논지 미리 제시)
2. 본론
   2-1. 핵심 개념 정의 / 텍스트 분석
   2-2. 논증 전개
   2-3. (필요 시) 예상 반론 검토
3. 결론 (논지 재정리 + 한계와 추가 논점)
참고문헌
```

### 1-2. 석사학위논문

| 항목 | 기준 |
|------|------|
| 분량 | 50–100쪽 |
| 구조 | 5장 구조 (학교마다 차이 있음) |
| 초록 | 국문·영문 초록 모두 필수 (학교 양식 따름) |

**일반적 5장 구조:**

```
Ⅰ. 서론 — 연구 목적, 필요성, 연구 문제, 연구 방법, 논문 구성
Ⅱ. 이론적 배경 / 선행연구 검토
Ⅲ. 연구 방법 (인문학에서는 "핵심 개념 분석 틀" 또는 "텍스트 분석 방법")
Ⅳ. 논의 / 분석
Ⅴ. 결론 — 연구 결과 요약, 교육적 시사점, 한계와 후속 연구 과제
참고문헌
부록 (필요 시)
```

> **주의**: 인문학 학위논문 5장 구조는 사회과학에서 차용된 형식이다. 순수 철학 논문은 5장 구조 대신 본문 장 수를 자유롭게 설정하기도 한다. 지도교수와 학과 양식을 우선 확인하라.

### 1-3. KCI 등재지 소논문

| 항목 | 기준 |
|------|------|
| 본문 분량 | 보통 200자 원고지 100–150매 (A4 약 20–30매) |
| 국문 초록 | 보통 600–800자 |
| 영문 초록(Abstract) | 보통 150–200단어 |
| 키워드 | 보통 5개 내외 (국문·영문 모두) |

> **주의**: 분량·초록·키워드 수는 학술지마다 다르다. 위는 한국 인문학 KCI 등재지의 일반적 범위일 뿐, **반드시 투고규정을 확인**하라.

### 1-4. 인문학 논문의 본문 흐름 (IMRaD 대안)

자연·사회과학의 IMRaD(Introduction–Methods–Results–Discussion)는 인문학 논문에 잘 맞지 않는다. 인문학 논문은 다음 흐름을 따른다:

```
서론 (문제 제기)
  ↓
선행연구 검토 (왜 기존 논의로는 부족한가)
  ↓
개념 / 텍스트 분석 (핵심 용어 정의, 1차 텍스트 독해)
  ↓
논증 (저자의 주장과 그 근거)
  ↓
예상 반론 검토와 재반박
  ↓
결론 (논지 재정리, 함의)
```

도덕윤리교육 전공 논문의 경우 결론 직전 또는 결론 안에 **"도덕교육적 함의" 또는 "교육적 시사점"** 절을 두는 것이 관례다.

## 2. 한국 인문학 통용 인용 표기 (각주 방식)

> **주의**: 아래는 이화여대 한국문화연구원 등에서 사용하는 한국 인문학 표준 형식이다. **한국철학회처럼 저자–연도–쪽수만 적는 학술지도 있다(아래 2-7 참조)**. 투고규정을 반드시 확인하라.

### 2-1. 학술지 논문

```
저자, 「논문제목」, 『학술지명』 권(호), 발행처, 발행연도, ○쪽.
```

예시:
```
홍길동, 「아리스토텔레스의 아크라시아 개념 재고」, 『도덕윤리과교육』 75호, 한국도덕윤리과교육학회, 2022, 35쪽.
```

### 2-2. 단행본

```
저자, 『책제목』, 출판사, 출판연도, ○쪽.
```

예시:
```
김상봉, 『도덕교육의 파시즘』, 길, 2005, 120쪽.
```

### 2-3. 번역서

```
원저자, 역자 역, 『한국어 제목』, 출판사, 연도, ○쪽.
```

예시:
```
아리스토텔레스, 강상진·김재홍·이창우 역, 『니코마코스 윤리학』, 길, 2011, 280쪽.
```

> 원어 제목 병기가 필요하면 한국어 제목 뒤 괄호 안에 표기: `『니코마코스 윤리학』(Ἠθικὰ Νικομάχεια)`

### 2-4. 단행본 수록 논문(편저 챕터)

```
저자, 「논문제목」, 편자 편, 『책제목』, 출판사, 연도, ○쪽.
```

### 2-5. 재인용

원출처를 본인이 직접 확인하지 못하고 다른 논문/책을 거쳐 인용한 경우 반드시 표기한다.

```
원저자, 「원논문」, 『원학술지』 호수, 연도, ○쪽; 인용한 저자, 『인용한 책』, 출판사, 연도, ○쪽에서 재인용.
```

> **금지**: 원출처를 보지 않고 직접 인용한 것처럼 표기하면 연구 부정행위로 간주된다.

### 2-6. 같은 자료 반복 인용 — 약어 사용

| 약어 | 의미 | 한국식 표현 |
|------|------|-------------|
| `Ibid.` (또는 `같은 책`) | 바로 앞 각주와 동일 자료 | "위의 책" / "위의 글" |
| `op. cit.` (또는 `앞의 책`) | 앞에서 인용한 자료 (다른 자료 끼인 후) | "앞의 책" / "앞의 글" |

예시:
```
1) 김상봉, 『도덕교육의 파시즘』, 길, 2005, 120쪽.
2) 위의 책, 125쪽.                              ← 바로 앞과 같은 책
3) 박이문, 『철학이란 무엇인가』, 일조각, 2001, 50쪽.
4) 김상봉, 앞의 책, 130쪽.                       ← 1번 책 다시 인용
```

> **주의**: 한국철학회 등 일부 학술지는 `같은 책`, `위의 책`, `Ibid.`를 **사용하지 않는다**. 같은 자료라도 매번 저자명+연도+쪽수를 반복 표기한다. 투고규정 우선.

### 2-7. 한국철학회식(저자–연도) 각주

각주에 약식 정보만 적고 자세한 서지는 참고문헌 목록에서 밝히는 방식이다.

```
국문: 김재권 (2007), 35쪽.
영문: Kim (1998), p. 78.
```

여러 문헌을 함께 인용할 때는 세미콜론으로 연결한다:
```
김재권 (2007), 35쪽; 박이문 (2001), 50쪽.
```

동일 저자의 동일 연도 문헌은 `2007a`, `2007b`로 구분한다.

### 2-8. 참고문헌 정렬 순서

1. **한국어 문헌** (저자명 가나다순)
2. **동양어 문헌** (한자/일본어 등, 한자 부수순 또는 가나다순)
3. **서양어 문헌** (저자 성(surname) 알파벳순)

각 그룹 안에서:
- 동일 저자의 여러 문헌은 발행연도 순
- 동일 저자·동일 연도는 `2007a`, `2007b`로 구분

### 2-9. 부호 체계 (한국 인문학 관례)

| 대상 | 동양어 | 서양어 |
|------|--------|--------|
| 단행본·학술지명 | `『 』` | *italics* |
| 논문·작품명 | `「 」` | `" "` |
| 작품 내 짧은 인용 | `" "` | `" "` |

## 3. Chicago Notes-Bibliography 스타일 (영문 인용 시)

영문 자료를 인용할 때 또는 학술지가 Chicago를 요구할 때 사용한다. 인문학에서는 **Notes-Bibliography(NB)** 시스템이 표준이며, **Author-Date** 시스템과 다르므로 혼동하지 말 것.

### 3-1. 책

**첫 노트 (Full Note):**
```
1. Charles Yu, Interior Chinatown (Pantheon Books, 2020), 45.
```

**짧은 노트 (Short Note) — 같은 자료 두 번째 인용부터:**
```
2. Yu, Interior Chinatown, 48.
```

**Bibliography 항목:**
```
Yu, Charles. Interior Chinatown. Pantheon Books, 2020.
```

### 3-2. 학술지 논문

**첫 노트:**
```
1. Hyeyoung Kwon, "Inclusion Work: Children of Immigrants Claiming Membership
   in Everyday Life," American Journal of Sociology 127, no. 6 (2022): 1842–43,
   https://doi.org/10.1086/...
```

**짧은 노트:**
```
2. Kwon, "Inclusion Work," 1851.
```

**Bibliography:**
```
Kwon, Hyeyoung. "Inclusion Work: Children of Immigrants Claiming Membership
   in Everyday Life." American Journal of Sociology 127, no. 6 (2022): 1818–59.
   https://doi.org/10.1086/...
```

### 3-3. 번역서

**첫 노트:**
```
1. Liu Xinwu, The Wedding Party, trans. Jeremy Tiang (Amazon Crossing, 2021), 279.
```

**Bibliography:**
```
Liu Xinwu. The Wedding Party. Translated by Jeremy Tiang. Amazon Crossing, 2021.
```

### 3-4. 17판 ibid. 정책 변화

> **중요 변경**: Chicago 17판은 반복 인용에서 **`ibid.` 사용을 권장하지 않는다**. 대신 **짧은 노트(short note)** 형식을 사용하라.

이전 (16판까지 허용):
```
1. Yu, Interior Chinatown, 45.
2. Ibid., 48.
```

권장 (17판 이후):
```
1. Yu, Interior Chinatown, 45.
2. Yu, Interior Chinatown, 48.
```

### 3-5. Bibliography 형식 규칙

- 알파벳순 정렬 (저자 성 기준)
- 두 번째 줄부터 0.5인치(또는 5칸) **hanging indent**
- 본문 종료 후 별도 페이지

## 4. 철학 논문 특수 사항

### 4-1. 1차 텍스트 표준 페이지(Bekker 등)

고전 철학 텍스트는 어떤 판본·번역본을 보든 공통으로 참조할 수 있는 **표준 페이지 번호**가 있다. 본문/각주 인용 시 표준 번호를 우선 사용한다.

| 저자 | 표준 페이지 시스템 | 예시 |
|------|------------------|------|
| 아리스토텔레스 | Bekker 번호 | `NE VII.3, 1147a24–b5` |
| 플라톤 | Stephanus 번호 | `Republic 514a–517a` |
| 칸트 | Akademie-Ausgabe (AA) | `KrV A51/B75` (순수이성비판은 A/B 판) |
| 토마스 아퀴나스 | Summa 구조 | `ST I-II, q. 77, a. 2` |

> 자세한 1차 텍스트 인용 규칙은 별도 스킬 `aristotle-primary-citation`을 참조.

### 4-2. 본문 안 그리스어/라틴어 표기

원어 단어를 본문에 노출할 때는 **이탤릭** 처리한다.

```
좋음: 아리스토텔레스가 말하는 *akrasia*(자제 없음)는 단순한 동물적 충동이 아니다.
좋음: 의지의 약함(*akrasia*)을 단순히 *enkrateia*(자제)의 결여로 환원할 수 없다.
```

> 한글 음역(아크라시아)을 먼저 쓰고 괄호 안에 원어를 넣는 방식이 학부 논문에서 더 친절하다.

### 4-3. 개념어 첫 등장 시 원어 병기

핵심 개념어는 **첫 등장 시 한 번만** 원어를 병기하고, 이후에는 한국어만 사용한다.

```
좋음: 아크라시아(akrasia, ἀκρασία)는 ... 이후 본문에서는 "아크라시아"로만 표기.
나쁨: 매번 "아크라시아(akrasia)"를 병기.
```

### 4-4. 직접 인용 vs 간접 인용

| 구분 | 표기 방법 |
|------|-----------|
| 직접 인용 (3행 미만) | 본문에 큰따옴표 `" "`로 묶음 |
| 직접 인용 (3행 이상) | 별도 단락으로 분리, 양쪽 들여쓰기, 따옴표 생략 |
| 간접 인용 (요약·바꿔쓰기) | 따옴표 없이 본문에 통합, 각주는 동일하게 표기 |

직접 인용이든 간접 인용이든 **반드시 각주를 단다**. 간접 인용이라고 출처를 누락하면 표절이다.

## 5. akrasia 주제 학부 논문 작성 예시 흐름

도덕윤리교육 학부 텀페이퍼 가정. A4 12매 분량.

```
[제목] 아리스토텔레스 아크라시아 개념의 도덕교육적 함의

1. 서론
   - 문제 제기: 학생들이 "옳다고 알면서도 실천하지 않는" 현상
   - 논지 미리 제시: 아리스토텔레스의 아크라시아 분석이 도덕교육에 주는 시사점
   - 논문 구성 안내

2. 아크라시아 개념의 정의
   2-1. 어원과 일상적 용법
   2-2. 『니코마코스 윤리학』 VII권에서의 정의 (1145b8 이하)
   2-3. enkrateia(자제)와의 대비

3. 아크라시아의 발생 메커니즘 — NE VII.3 분석
   3-1. 소크라테스적 입장 비판 (1145b21–27)
   3-2. 실천적 추론(practical syllogism)에서의 단절
   3-3. 욕구가 지식을 가리는 방식

4. 도덕교육적 함의
   4-1. "지행합일" 교육 모델의 한계
   4-2. 습관화(ethismos) 교육의 중요성
```

## 6. 흔한 실수와 회피 방법

### 6-1. 각주 번호와 참고문헌 항목 불일치

각주에 등장한 모든 자료는 참고문헌 목록에 있어야 하고, 그 반대도 성립해야 한다(일부 학술지는 "읽었으나 직접 인용하지 않은 자료"를 참고문헌에 포함하기도 함 — 투고규정 확인).

**점검 방법**: 탈고 직후 각주 번호 1번부터 끝까지 훑으며, 등장하는 모든 저자·서지를 참고문헌 목록과 1:1 대조.

### 6-2. 페이지 범위 표기 불일치

한 논문 안에서 다음 두 형식이 섞이면 안 된다:

| 일관 사용 | 혼용 금지 |
|-----------|-----------|
| `35쪽` 또는 `pp. 35–48` | `p. 35`와 `35쪽`을 같은 논문에 혼용 |
| `35–48` 또는 `35-48` (en-dash 또는 hyphen) | 둘을 섞어서 사용 |

> 한국어 자료는 `쪽`, 영문 자료는 `p.`/`pp.`로 통일하는 방식이 가장 흔하다.

### 6-3. 직접 인용 vs 간접 인용 구분 누락

- 원문을 그대로 옮기면서 따옴표를 빠뜨리면 **표절**
- 바꿔 쓴 후 출처를 누락해도 **표절**
- 직접 인용은 토씨까지 정확히, 변형이 있으면 `[ ]`로 표시

### 6-4. 1차 텍스트를 번역본 페이지로만 인용

```
나쁨: 아리스토텔레스, 『니코마코스 윤리학』, 길, 2011, 280쪽.
좋음: 아리스토텔레스, 『니코마코스 윤리학』 1147a24–b5 (강상진 외 역, 길, 2011, 280쪽).
```

표준 페이지를 함께 적어야 다른 판본을 가진 독자도 검증할 수 있다.

### 6-5. 학술지명·논문명 부호 오용

- 책·학술지 → `『 』`
- 논문·장(章) → `「 」`
- 작품(영화·그림 등) → `< >` 또는 `〈 〉`

부호를 바꿔 쓰면 편집자가 가장 먼저 지적한다.

### 6-6. 같은 저자 동일 연도 문헌 미구분

저자–연도 방식 인용에서 같은 저자의 같은 해 문헌이 둘 이상이면 반드시 `2007a`, `2007b`로 구분.

### 6-7. URL/DOI 누락

학술지 논문은 가능하면 DOI를, 웹 자료는 접속 일자와 URL을 명시. Chicago 17판은 DOI를 우선 표기.

## 7. 작성 전 체크리스트

- [ ] 투고하려는 학술지의 투고규정을 직접 다운받아 읽었는가
- [ ] 분량·초록 단어 수·키워드 개수가 규정에 맞는가
- [ ] 각주 형식(서지 전체 vs 저자-연도 약식)이 학술지 규정과 일치하는가
- [ ] 참고문헌 정렬(국문→동양→서양)을 지켰는가
- [ ] 1차 텍스트는 표준 페이지(Bekker 등)를 병기했는가
- [ ] 각주 번호와 참고문헌 항목이 1:1로 일치하는가
- [ ] 직접 인용에 따옴표 또는 들여쓰기를 적용했는가
- [ ] 재인용한 자료에 "재인용" 표기를 했는가
- [ ] 부호 `『』`/`「」`/이탤릭 사용이 일관되는가
