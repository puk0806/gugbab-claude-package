# YMYL 콘텐츠 SEO

> 소스:
> - Google Search Quality Rater Guidelines (2025-09-11 개정) — https://guidelines.raterhub.com/searchqualityevaluatorguidelines.pdf
> - schema.org MedicalWebPage / MedicalCondition / LegalService / InvestmentOrDeposit

## 1. YMYL이란

**YMYL = Your Money or Your Life**. 사용자의 *건강·재정·안전·사회적 후생*에 잠재적으로 영향을 미치는 토픽을 가리키는 Google 분류다.

### 1.1 Google QRG의 YMYL 범주 (2025-09-11 개정 기준)

| 범주 | 예시 |
|------|------|
| **Health or Safety** | 진단·치료·약물·정신건강·응급처치·식품 안전·자해/자살 |
| **Financial Security** | 투자·세금·대출·보험·은퇴·암호화폐·부동산 |
| **Government, Civics & Society** | 선거·투표·시민권·법률·이민·복지 정책 |
| **News and Current Events** | 자연재해·정치·국제 사건 (사실관계가 사회 후생에 영향) |
| **Shopping** | 고비용 의사결정(자동차·가전·교육 서비스) |

> 주의: YMYL은 **스펙트럼**이다. "명확한 YMYL", "명확히 비-YMYL", "중간"의 세 영역으로 판정한다. "내 토픽이 YMYL인가?"가 모호하면 *영향 잠재성* 기준으로 판정한다.

### 1.2 "Potential to Harm" 평가 축

QRG는 토픽이 사람에게 미칠 *잠재적 해(harm) 정도*로 분류한다:

1. **개인 신체·정신 건강** (자해, 약물 오남용)
2. **개인 재정 안정** (사기 투자, 잘못된 세금 정보)
3. **타인의 안전** (응급처치 오정보, 백신 허위정보)
4. **사회·집단의 후생** (선거 허위정보, 공중보건)

해의 잠재성이 높을수록 *highest E-E-A-T standards* 적용.

## 2. YMYL에서 E-E-A-T 더 엄격 적용

Google QRG는 YMYL 페이지에 대해 명시적으로 *"highest E-E-A-T standards"*를 요구한다. 일반 콘텐츠 E-E-A-T(→ `[[content-eeat-quality]]`)와의 차이는 다음과 같다.

| 신호 | 일반 콘텐츠 | YMYL 콘텐츠 |
|------|------------|------------|
| 저자 자격 | 도메인 경험 | **공인 자격증·면허번호** (의사·변호사·CFP) |
| 검토 프로세스 | 편집자 검토 | **자격 보유 전문가의 사실관계 검토** + 리뷰 라벨 노출 |
| 출처 | 신뢰 가능한 2차 출처 | **1차 출처**(정부 기관·학회·peer-reviewed) 비율 높을 것 |
| 최신성 | 정확성 유지 | **lastReviewed/dateModified** 명시, 정기 갱신 사이클 |
| 면책 | 권장 | **필수** — "일반 정보, 개인 진단/조언 아님" |
| 광고 구분 | 권장 | **필수 + 강화 표기** (제휴·후원·심의필번호) |

## 3. YMYL 신뢰 강화 7대 신호

### 3.1 저자 자격 명시
- 저자명 + 자격(의사면허·CFP·변호사 자격번호) + 소속 기관
- Author 페이지에 학력·임상 경력·논문 링크
- 익명 저자 또는 "편집팀" 단독 표기는 YMYL에서 신뢰도 크게 감점

### 3.2 전문가 검토 라벨
- `Medically reviewed by Dr. {이름}, {자격} on {YYYY-MM-DD}`
- Healthline 방식의 명시적 리뷰 byline이 표준. 본문 작성자와 리뷰어가 다를 경우 두 사람 모두 노출
- 검토 사이클: 임상 가이드 변경이 잦은 토픽은 6~12개월, 안정 토픽은 12~24개월

### 3.3 1차 출처 인용
- 의료: WHO·CDC·질병관리청·식약처·peer-reviewed (PubMed)
- 금융: 금감원·한국은행·국세청·공식 약관·재무제표
- 법률: 법제처 국가법령정보센터·대법원 판례·법원도서관
- 2차 출처(블로그·뉴스 요약)만으로 구성된 YMYL은 위험 신호

### 3.4 마지막 검토일 (lastReviewed / dateModified)
- 본문 상단에 가시 표기 + schema.org 마크업 양쪽
- "published"와 "reviewed"는 구분 표기

### 3.5 편집 가이드라인 공개
- About 페이지 또는 별도 `/editorial-policy` 페이지
- 사실관계 검증 절차, 출처 우선순위, 정정 정책, AI 활용 정책 명시
- 광고와 편집의 분리 정책 포함

### 3.6 광고 명확 구분
- 본문 상단 "광고 / 협찬 / 제휴" 라벨
- 어필리에이트 링크 별도 표기 (공정위 2024-06 시행 「추천·보증 등에 관한 표시·광고 심사지침」 준수)
- 의료·금융은 *심의필번호* 노출 (한국 한정)

### 3.7 면책 조항
- 의료: "본 정보는 일반 건강 정보이며 진단·치료를 대체하지 않습니다"
- 금융: "본 정보는 일반 정보이며 특정 상품 권유가 아닙니다. 투자에는 원금 손실 위험이 따릅니다"
- 법률: "본 정보는 일반 법률 정보이며 구체 사건에는 변호사 상담을 받으십시오"

## 4. 의료 콘텐츠 특화

### 4.1 한국 의료법 핵심 (의료법 제56조)
- **광고 주체 제한**: 의료기관 개설자·의료기관의 장·의료인이 아닌 자는 의료 광고 불가
- **금지 광고 유형**:
  - 신의료기술 평가 미경유 광고
  - 환자 치료 경험담 (치료 효과 오인 우려)
  - 객관적 사실과 다른 거짓 광고
  - 타 의료인과의 우월 비교 광고
  - 심각한 부작용 등 중요 정보 누락 광고

### 4.2 한국 의료광고 사전심의 (2025년 강화)
- 사전심의 주체: 대한의사협회·대한치과의사협회·대한한의사협회 자율심의기구
- **2025년 가이드라인 개정**: 블로그·유튜브·인스타그램이 원칙적으로 사전심의 대상에 포함. 네이버처럼 일평균 방문자 10만 명 초과 플랫폼의 모든 의료 게시물이 대상으로 해석됨
- 심의필번호 본문 표기 필수

### 4.3 의료 콘텐츠 안전 안내
- 자해·자살 키워드 글은 본문 상단/하단에 *위기 자원* 안내 박스 필수
  - **109** (자살예방 통합 상담, 2024-01-01 개통)
  - **1577-0199** (정신건강 위기상담)
  - **119** (응급상황)
- 자세한 안내 문구·다국어 자원 → `[[crisis-intervention-resources-korea]]`
- 자살예방 보도준칙 4.0 준수: 구체적 방법·도구·장소·동기 비공개

### 4.4 미신·민간요법 vs 의학 근거 정보
- 민간요법을 다룬다면 **반드시** 의학적 근거 평가와 함께 (Cochrane Review·KMLE 인용)
- "효과 있다" 단정 금지 → "현재 임상 근거는 X 수준"
- 부작용·금기 사항 누락 금지

### 4.5 자율 인증
- HONcode는 2022년 운영 종료. 대안: NHS Information Standard, Health on the Net 후속 인증, peer-reviewed 사이트 인증
- 한국은 정부 공인 의료정보 인증 제도가 부재. *자체* 편집 가이드라인 공개가 차선

## 5. 금융 콘텐츠 특화

### 5.1 한국 금융광고 규제 핵심
- **금융소비자보호법** (금소법, 2021-03-25 시행) 제22조: 금융상품 광고 규제
- 광고 사전심의: 보험(생명/손해보험협회)·대부(대부금융협회) 등 권역별
- **광고심의필번호** 광고물에 표시 필수
- 보험 광고: 보장 내용 누락·과장 금지

### 5.2 필수 표기 요소
- **원금 손실 가능성**: 투자상품은 "투자 결과에 따라 원금 손실 가능"
- **과거 수익률 ≠ 미래 보장**: "과거 수익률은 미래 수익을 보장하지 않습니다"
- **수수료·세금**: 총비용 명시
- **계약 전 알릴 의무 사항** 링크
- **광고 유효기간** (보험)

### 5.3 어필리에이트 표기 (공정위 2024-06 시행)
- 신용카드·대출·증권 계좌 추천 시 **수수료 수령 사실 명시**
- 본문 상단 + 해당 링크 직전 양쪽 표기 권장
- "이 글은 제휴(어필리에이트) 링크를 포함합니다" 수준의 *명확한* 한국어 고지

### 5.4 피해야 할 표현
- "안전한 고수익" — 모순 표현 (금소법 위반 소지)
- "원금 보장 + 고수익" — 금감원 단속 대상
- "OO 전문가 추천" 단정 — 검증 가능한 자격 명시 안 하면 위험
- 가격·이율을 *현재 시점*만 표기하고 변경 시점 표기 누락

## 6. 법률 콘텐츠 특화

### 6.1 핵심 원칙: 일반 정보 vs 법적 조언
법률 콘텐츠는 *법률 정보 제공*과 *변호사-의뢰인 관계 법적 조언*을 명확히 분리해야 한다.

- 본문 도입부: "본 글은 일반 법률 정보이며 구체적 법적 조언이 아닙니다"
- 결론부: "구체 사건의 판단·대응은 변호사 상담을 받으시기 바랍니다"
- 변호사가 작성한 경우에도 "변호사-의뢰인 관계가 성립하지 않는다" 표기

### 6.2 한국 변호사 광고 규정 (2025-02-06 개정)
- 규범 체계: 변호사법 + 「변호사 광고에 관한 규정」 + 「변호사 광고에 관한 규칙」 (2024-10 신설) + 변호사검색서비스 운영 가이드라인 (2025-05)
- **AI 사용 표기 의무**: 변호사가 AI를 사용한다고 광고하려면 AI 시스템을 협회에 사전 등록, AI 결과물을 검토한 변호사 이름 명시
- **공직 경력 광고 제한**: "전관 변호사", "전관예우" 단어 사용 금지
- 변호사 아닌 자의 법률 광고는 변호사법 제109조 위반 소지

### 6.3 출처 인용
- 법령: 국가법령정보센터 (law.go.kr) — 조문 번호·시행일 명시
- 판례: 대법원 판결문 (court.go.kr 판례검색·법원도서관) — 사건번호 + 선고일
- 행정해석·유권해석: 발신 기관·문서번호
- "한 판례에서는" 식의 *무출처* 인용은 YMYL에서 신뢰도 치명적

### 6.4 피해야 할 표현
- "100% 승소 가능" — 변호사 광고 규정 위반
- 비교 광고 ("타 로펌보다 우수") — 원칙적 금지
- 특정 결과 보장 ("벌금 → 무죄로")

## 7. YMYL에서 피해야 할 패턴 (Anti-Patterns)

| 패턴 | 위험 |
|------|------|
| **AI 양산 + 사람 검토 없음** | Google "scaled content abuse" 정책 위반 (2024-03 정책 강화). 의료·금융은 manual action 우선 타겟 |
| **익명 저자** | 자격 검증 불가. YMYL에서 사실상 자동 감점 |
| **출처 없는 통계** | "한 연구에 따르면 80%가…" — 출처 미기재 통계 |
| **"기적의 치료" 단정** | "100% 효과", "부작용 없음" 등 단정 표현 |
| **의사·변호사 흉내** | 자격 없이 "처방" "법적 조언" 단정형 어조 |
| **민간요법 무비판 소개** | 근거 평가 없이 효능 소개 |
| **응급 키워드 + 안전 안내 누락** | 자해·자살·약물 과량 키워드 글에 109/1577-0199 누락 |
| **광고와 편집 미분리** | "추천 상품" 형식의 광고를 일반 정보로 위장 |
| **dateModified 누락 또는 가짜 갱신** | 본문 변경 없이 날짜만 바꾸는 *fake freshness* |

## 8. YMYL에서 안전한 AI 활용

YMYL에서 AI 사용 자체가 금지는 아니지만, *editorial oversight* 없는 AI 양산은 Google의 scaled content abuse 정책 직격 대상이다.

### 8.1 안전한 워크플로우
```
AI 초안 생성
  → 도메인 전문가(의사·CFP·변호사) 검토
  → 사실관계 1차 출처 대조
  → 전문가 서명 + 검토일 기록
  → "Medically/Legally/Financially reviewed by ..." 라벨 노출
  → 정기 재검토 사이클 설정
```

### 8.2 AI 사용 정책 공개
- 편집 가이드라인 페이지에 "AI 사용 정책" 섹션
- 어디서 AI를 쓰고 어디서 쓰지 않는지 명시
- 한국 변호사 광고 규정처럼 AI 사용 자체를 공시해야 하는 도메인 있음 (법률 분야)

### 8.3 사실관계 100% 외부 출처
- AI 생성 텍스트의 *모든* 사실관계 클레임은 외부 1차 출처로 검증
- 인용 출처를 그대로 본문에 노출 (각주 또는 인라인)

## 10. Helpful Content + YMYL

### 10.1 People-first 가중치 상승
Google Helpful Content System은 YMYL에서 *더 가혹하게* 적용된다:
- 키워드 stuffing 페널티 더 큼
- "AI가 생성한 듯한" 표현(예: "이 글에서는…에 대해 알아보겠습니다" 도입부 양산)에 민감
- 사용자 의도 충족도 더 엄격하게 측정

### 10.2 2024~2025 정책 변화
- **2024-03**: scaled content abuse 정책 강화. "자동 생성 여부 자체보다 *abusive behavior*"에 초점
- **2025-06**: 사이트 단위 manual action 본격화. 의료·금융 도메인 우선 타겟
- 콘텐츠 양 ≠ 가치. *editorial value-add* 없는 양산은 50~80% 트래픽 하락 보고됨

## 11. 한국 시장 특수성

### 11.1 네이버 검색 환경
- 한국 사용자의 의료·금융 검색은 네이버 점유율이 여전히 높음
- 네이버는 자체 의료/건강 콘텐츠 평가 알고리즘을 가짐 (세부 미공개). *의사/병원/공식 기관* 출처 우대 경향
- 네이버 인플루언서/블로그도 의료 사전심의 대상 (2025 가이드)
- Google + 네이버 양쪽 SEO를 *함께* 고려

### 11.2 한국어 콘텐츠의 출처 인용 빈약 문제
- 한국어 의료/금융/법률 블로그는 *출처 인용 비율이 영어권 대비 현저히 낮음*
- 의식적으로 1차 출처 인라인 인용 강화 → 동일 토픽에서 차별화 가능
- 출처 표기 권장: 본문 인라인 + 글 하단 "참고 자료" 섹션 양쪽

### 11.3 관련 한국 법규 한 줄 정리

| 도메인 | 법령 | 핵심 |
|--------|------|------|
| 의료 | 의료법 제56조 | 광고 주체·내용 제한, 사전심의 |
| 금융 | 금융소비자보호법 제22조 | 광고 사전심의, 필수표기 |
| 법률 | 변호사법 + 변호사 광고에 관한 규정 (2025-02) | AI 표기, 공직 경력 표기 제한 |
| 일반 광고 | 표시·광고의 공정화에 관한 법률 | 어필리에이트·후원·뒷광고 표기 (2024-06 강화) |

## 12. YMYL 점검 체크리스트

콘텐츠 발행 전 다음을 모두 확인한다.

### 12.1 신뢰성
- [ ] 저자 자격(면허번호·CFP·변호사 등록번호)이 본문 또는 byline에 표기
- [ ] 전문가 검토자 별도 표기 ("Medically reviewed by …")
- [ ] 1차 출처(정부·학회·peer-reviewed) 비율 ≥ 50%
- [ ] 모든 통계·수치에 출처 인라인
- [ ] dateModified/lastReviewed 본문 + schema 양쪽 표기

### 12.2 규정 준수
- [ ] 의료/금융: 해당 사전심의 통과·심의필번호 표기
- [ ] 법률: "일반 정보, 법적 조언 아님" 면책
- [ ] 광고·제휴 링크 명확 구분 (공정위 표시·광고 심사지침 준수)
- [ ] 변호사 광고: AI 사용 표기, 공직 경력 표기 규정 준수

### 12.3 안전
- [ ] 자해·자살 키워드 글: 109 + 1577-0199 + 119 안내 박스
- [ ] 약물·중독 글: 응급 연락처
- [ ] 임신·소아 글: 응급 안내
- [ ] 단정 표현("100% 효과") 제거

### 12.4 schema.org
- [ ] 의료: `MedicalWebPage` + `reviewedBy` Person + `lastReviewed`
- [ ] 금융: `Article` + `about: FinancialProduct/InvestmentOrDeposit`
- [ ] 법률: `Article` + author.worksFor `LegalService`
- [ ] Person `affiliation` 또는 `worksFor`로 소속 명시

## 13. 흔한 실수 패턴 (사례)

### 13.1 AI 양산 의료 사이트
- **증상**: 매일 수십 건 발행, 모두 비슷한 구조, 저자 "편집팀"
- **결과**: 2024-03 이후 manual action으로 deindex 사례 다수
- **수정**: 발행 빈도 1/10로 축소 + 의사 검토 라벨 + 1차 출처 인라인

### 13.2 익명 금융 블로그
- **증상**: 저자 정보 없음, "전문가"라는 표현만 반복
- **결과**: 금감원 위반 + Google 신뢰도 감점
- **수정**: 저자 실명·자격·소속 공개, About 페이지 작성, 어필리에이트 표기

### 13.3 "한 연구에 따르면" 무출처
- **증상**: 통계는 많은데 모두 출처 없음
- **결과**: AI 생성 의심 + 검증 불가로 신뢰도 감점
- **수정**: 모든 통계에 [출처: WHO 2024 보고서, p.34] 형식 인라인

### 13.4 한국 의료법 무시
- **증상**: 환자 치료 경험담 노출, "타 병원 대비 우수" 비교
- **결과**: 의료법 제56조 위반 → 행정 처분 + 검색 제재
- **수정**: 사전심의 통과 자료만 게시, 심의필번호 표기

### 13.5 자살 키워드 글 + 자원 안내 누락
- **증상**: 자살 통계·방법 언급 글에 위기 자원 안내 없음
- **결과**: 자살예방 보도준칙 위반 + 사회적 해 + 검색 신뢰도 감점
- **수정**: 본문 상단/하단에 109/1577-0199 안내 박스 필수 삽입. 자세한 안내문 → `[[crisis-intervention-resources-korea]]`

### 13.6 fake freshness
- **증상**: 본문 변경 없이 dateModified만 매월 갱신
- **결과**: Google 알고리즘이 *fake freshness* 패턴 감지
- **수정**: 실제 사실관계 갱신 시에만 dateModified 변경, 변경 내역 본문에 표기

## 14. 정리

YMYL 콘텐츠는 일반 SEO 위에 *전문가 검증·법규 준수·안전 안내* 3개 층을 더해야 한다.

핵심 우선순위:
1. **저자·검토자 자격 명시** (한 줄도 빠지면 안 됨)
2. **1차 출처 인용** (한국어 콘텐츠일수록 의식적으로 강화)
3. **dateModified/lastReviewed 정직하게**
4. **한국 도메인이면 의료법·금소법·변호사법 + 공정위 표시광고 심사지침 준수**
5. **자해·자살·응급 키워드는 위기 자원 안내 필수** (109, 1577-0199, 119)
6. **AI는 보조, 검토와 책임은 자격 보유 인간**
