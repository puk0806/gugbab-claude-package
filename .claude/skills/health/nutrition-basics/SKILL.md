## 3대 영양소 역할 및 특성

### 탄수화물
- **역할**: 주 에너지원, 뇌·신경계 포도당 공급
- **종류**: 단순당(과일·설탕), 복합당(곡류·채소), 식이섬유
- **식이섬유 권장량**: 남성 25g/일, 여성 20g/일 (충분섭취량)
- **권장**: 정제 탄수화물보다 통곡물·채소 위주

### 단백질
- **역할**: 근육·효소·호르몬·면역물질 구성
- **권장섭취량 산출**: 체중 kg × 0.91g (성인 평균)
- **1일 권장섭취량**
  - 남성 19-64세: 65g
  - 여성 19-64세: 55g
- **양질의 단백질 공급원**: 육류, 생선, 달걀, 콩류, 유제품

### 지방
- **역할**: 지용성 비타민(A·D·E·K) 흡수, 세포막 구성, 호르몬 합성
- **종류**: 포화지방(동물성), 불포화지방(식물성·생선), 트랜스지방(가공식품)
- **권장**: 불포화지방 위주, 포화지방·트랜스지방 최소화

## 칼로리 계산 방법

### 기초대사량(BMR) — Harris-Benedict 공식 (수정판)

```
남성: BMR = 88.4 + (13.4 × 체중kg) + (4.8 × 키cm) - (5.7 × 나이)
여성: BMR = 447.6 + (9.2 × 체중kg) + (3.1 × 키cm) - (4.3 × 나이)
```

### 활동 계수 (TDEE = BMR × 활동계수)

| 활동 수준 | 계수 |
|----------|------|
| 거의 없음 (좌식) | 1.2 |
| 가벼운 활동 (주 1-3일 운동) | 1.375 |
| 보통 활동 (주 3-5일 운동) | 1.55 |
| 매우 활동적 (주 6-7일 운동) | 1.725 |
| 극단적 활동 (운동선수 등) | 1.9 |

### 체중 목표별 칼로리 조정

| 목표 | 조정 |
|------|------|
| 체중 감량 | TDEE - 300~500 kcal/일 |
| 체중 유지 | TDEE |
| 근육 증가 | TDEE + 200~300 kcal/일 |
